調整卡片透明度，黑洞與水族箱背景不再被遮擋

你的主題背景不會再被卡片擋到看不見了。以前在使用「黑洞」或「午夜水族館」這兩個主題面板時，卡片本身的顏色太濃、毛玻璃效果太霧，導致黑洞的吸積盤，或者水族館裡游來游去的魚和水母，幾乎全被蓋住。現在系統大幅調降了卡片的透明度與模糊效果，讓背景裡的漂亮圖案能清楚透出來，同時也確保卡片上的字一樣好讀。此外，專案的 README 說明書也參考了各大優秀開源工具的標準重新排版，並補齊了所有最新功能的介紹。就像你家新買了一個超美的魚缸當隔間牆，結果原本裝的毛玻璃太厚，什麼魚都看不到；現在換成了更清透的玻璃，不但保留了隔間的隱私（字一樣好讀），還能清楚欣賞魚群游動。

Your gorgeous themed backgrounds will no longer be completely obscured by the data cards on top of them. Previously, when you used the "Black Hole" or "Midnight Aquarium" panels, the card opacity was so heavy and the background blur so intense that the black hole's accretion disk, or the aquarium's fish and jellyfish, were virtually buried. The system has now significantly lowered both the opacity and the blur on these cards, allowing the beautiful canvas art to shine through clearly while keeping the text completely readable. Additionally, the project's README manual has been thoroughly restructured following best practices from top open-source tools, and now includes a complete inventory of all current features. It's like installing a stunning aquarium as a room divider, only to realize the frosted glass you used was so thick you couldn't see the fish; now you've swapped it for crystal-clear glass, keeping the separation of the room intact while letting you admire the vibrant marine life.

카드 투명도를 조절해 블랙홀과 수족관 배경이 가려지지 않고 잘 보인다.

테마의 멋진 배경이 카드에 가려 안 보이는 일이 사라졌다. 예전엔 '블랙홀'이나 '자정의 수족관' 테마 패널을 쓸 때 카드의 색이 너무 진하고 불투명 유리 효과가 짙어서, 블랙홀의 강착 원반이나 수족관 안에서 헤엄치는 물고기와 해파리가 거의 다 가려졌다. 이제 시스템이 카드의 투명도와 흐림 효과를 크게 낮춰 배경의 예쁜 그림이 선명하게 비치게 만들면서도 글자는 여전히 읽기 편하게 유지했다. 또한, 프로젝트의 README 설명서도 잘 관리되는 훌륭한 오픈소스 도구들의 기준을 참고해 완전히 새로 구성하고 모든 최신 기능 소개를 빠짐없이 채워 넣었다. 방을 나누려고 엄청 예쁜 수족관을 샀는데 그 앞을 너무 두꺼운 불투명 유리로 막아둬서 물고기를 하나도 볼 수 없었던 것을, 훨씬 투명한 유리로 바꿔 방을 나누는 역할(글자 읽기)은 그대로 두면서 물고기가 헤엄치는 모습은 선명하게 감상할 수 있게 한 것과 같다.
