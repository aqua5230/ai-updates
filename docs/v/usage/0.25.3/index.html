zh-TW

省額度模式更持久，精靈動畫常駐陪伴

「Token Saver」（原本叫 Terse Mode，為了讓你一聽就懂它的好處而改名）現在不管你聊多久，都能持續發揮省額度的效果。以前跟 Claude 聊到後來，它會忘記開頭的約定，回覆越寫越長（實測字數暴增 84%）；現在只要開啟這個模式，系統會在你每次講話時，偷偷在背後附上一句簡短的提醒，讓 Claude 乖乖保持簡潔，成功把後期的廢話縮短了約 40%。這項功能在更新後會自動生效（僅限 Claude Code）。此外，選單列上的鳳凰和龍精靈動畫現在變成永久常駐了，原本的開關已經被移除，牠們會一直陪伴著你。就像你請了一個隨身秘書，以前他只在出門前提醒你「長話短說」，結果聊久了你還是會忘記，現在他會在你每次開口前都捏一下你的手肘提醒你；而且這個秘書現在會整天跟著你，不會再下班消失了。

Original changelog
**Token Saver now holds up in long conversations**: enabling it also installs a per-message reminder hook (`usage_terse_reminder.py`, UserPromptSubmit) that re-injects a one-line terse nudge with every message you send. In an A/B test on real Claude sessions the start-of-session instruction alone drifted badly — late-conversation replies grew +84% (603 → 1108 chars) — while the tail reminder held them steady (538 → 674), keeping late replies ~40% shorter. Existing installs pick the new hook up automatically via self-heal. Claude Code only (Codex CLI has no UserPromptSubmit equivalent).
**"Terse Mode" is renamed "Token Saver"** in the menu across all five languages — the name now states the benefit instead of the mechanism.
**Spirit critters are now always on**: the phoenix/dragon animation lives permanently beside the percentages; the "Summon Spirits / Dismiss Spirits" toggle is gone.

en

Token Saver stays strict all conversation long, and spirits are here to stay

"Token Saver" (renamed from "Terse Mode" so the benefit is obvious at a glance) now keeps saving your quota no matter how long the conversation runs. Previously, Claude would forget the initial instruction deep into a session and its replies would balloon in length (up 84% in real tests); now, enabling this installs a hook that quietly injects a terse nudge with every single message you send, successfully keeping late-conversation replies about 40% shorter. Existing installations will pick this up automatically (Claude Code only). Additionally, the phoenix and dragon spirit animations are now always on—the toggle to dismiss them is gone, and they will permanently accompany your percentages. It's like hiring an assistant who used to only tell you "keep it brief" at the start of a meeting, letting you ramble as time went on, but now quietly nudges you under the table every time you speak; and that assistant now stays by your side full-time instead of clocking out.

ja

節約モードが長話でも効果を持続し、精霊アニメーションも常駐化

「Token Saver」（直感的にメリットが分かるよう Terse Mode から改名）が、どれだけ長話になってもクォータ節約効果を発揮し続けるようになった。以前は Claude と長く話していると冒頭の指示を忘れ、返答がどんどん長引いていた（実測で文字数が 84% 増加）が、今はこのモードをオンにすると、あなたがメッセージを送るたびに裏側で「簡潔に」という念押しをこっそり添えてくれるようになり、終盤の無駄話を約 40% 短縮することに成功した。更新すれば既存の環境にも自動で適用される（Claude Code のみ）。また、メニューバーの不死鳥と竜の精霊アニメーションが常時表示されるようになり、非表示スイッチは削除され、ずっとあなたに寄り添うようになった。秘書が以前は会議の冒頭に「手短に」と言うだけで時間が経つとだらだら話すのを止められなかったのが、今はあなたが発言するたびに机の下でこっそりつつき、しかもその秘書が定時で帰らず常にそばにいてくれるようになったようなものである。

ko

할당량 절약 모드가 긴 대화에서도 끝까지 유지되며, 정령 애니메이션이 항상 곁을 지킨다.

'Token Saver'(듣자마자 장점을 알 수 있도록 Terse Mode에서 이름 변경)가 이제 대화가 아무리 길어져도 할당량 절약 효과를 계속 발휘한다. 예전엔 Claude와 오래 얘기하다 보면 맨 처음의 약속을 잊고 답변이 갈수록 길어졌는데(실제 테스트 결과 글자 수가 84% 폭증), 이제 이 모드를 켜면 당신이 말할 때마다 시스템이 백그라운드에서 짧게 답하라는 넛지를 몰래 끼워 넣어 후반부의 쓸데없는 말을 약 40%나 성공적으로 줄여준다. 업데이트하면 기존 설치 환경에도 알아서 적용된다(Claude Code 전용). 또한 메뉴 바의 불사조와 용 정령 애니메이션이 이제 항상 켜져 있도록 바뀌어 끄는 스위치가 사라지고 언제나 당신과 함께한다. 회의를 시작할 때만 '짧게 하라'고 귀띔해서 시간이 지나면 결국 말을 길게 하도록 내버려두던 비서가, 이제는 당신이 입을 열 때마다 책상 밑에서 쿡쿡 찔러주고, 퇴근하지 않고 항상 당신 곁을 지키게 된 것과 같다.
