zh-TW

報表換上暖色玻璃質感，還能一鍵存成分享圖片

你的數據報表現在變得更漂亮、也更好分享了。視覺上全面改版，換成了帶有琥珀色與翡翠色光暈的暖墨色背景，加上微微雜訊的毛玻璃（glassmorphism）卡片質感；整體的強調色也從舊版的藍紫色系，改成了金、翡翠、珊瑚色，字體大小也稍微調大、層次更分明。分享功能方面，現在多了一個「存成 .png 圖片」的按鈕，系統會在完全離線（不連網）的狀態下幫你把報表轉成圖片；如果你開啟了「隱藏專案名稱」的開關，圖片也會自動幫你把機密名字碼掉。此外，分享對話框的標題也改得更中性，不再假設你要傳給老闆或同事。在幕後，系統也順手把各項安全防護與自動化檢查（CI）升級到了新版本。就像是原本用普通紙印的黑白對帳單，現在升級成了燙金邊的霧面磨砂邀請函，而且不僅自己看著賞心悅目，要拍照傳給朋友時，還能一鍵自動幫你把私密資訊打上馬賽克。

Original changelog
**Save the report as a .png image**: the share dialog gains a third button next to .html/.csv. It renders the report with a vendored html-to-image v1.11.13 (MIT) — fully offline, no network, and it honors the "hide project names" toggle so a masked screenshot is one click.
**Report visual redesign (warm glassmorphism)**: warm ink background with amber/jade glows, frosted-glass cards over a subtle noise texture, and a gold/jade/coral accent palette replacing the old blue/purple. The contribution heatmap turns gold, the base type scale grows to 17.5px with a clearer size hierarchy, and headings/body adopt Grenette/Styrene font stacks (system-font fallbacks; no font files embedded or fetched).
**Share dialog copy is audience-neutral**: the "Send to a colleague / manager" heading is now "Share a copy" across all five locales.
CI hardening while closing out this cycle's supply-chain checklist: ClusterFuzzLite + Atheris fuzzing for the JSONL parsers (base image pinned by digest), CodeQL actions bumped to v4, and routine action bumps (setup-python 6.3.0, setup-uv 8.3.2) — all dependabot PRs merged and both OpenSSF Scorecard alerts triaged.

A warm glassmorphism redesign for the report, now exportable as a PNG image

Your data report is now much more beautiful and easier to share. The entire visual experience has been redesigned: it now features a warm ink background with amber and jade glows, frosted-glass (glassmorphism) cards over a subtle noise texture, and an accent palette of gold, jade, and coral that replaces the old blue and purple. The typography has also been scaled up slightly for clearer hierarchy. For sharing, the dialog now includes a new button to save the report as a `.png` image, running entirely offline with no network connection required; if you toggle "hide project names," the resulting screenshot automatically masks your confidential data. The dialog's wording is also more neutral now, no longer assuming you are sending it to a manager or colleague. Behind the scenes, the system also hardened various automated security checks (CI) to their latest versions. It's like taking a standard black-and-white bank statement and upgrading it into a gold-foil frosted-glass invitation—not only is it gorgeous to look at, but if you want to snap a photo to show a friend, a single click automatically blurs out your account numbers.

ko

보고서가 따뜻한 유리 질감으로 바뀌고 클릭 한 번에 이미지로 저장된다.

데이터 보고서가 훨씬 예뻐지고 공유하기도 편해졌다. 디자인이 전면 개편되어, 호박색과 비취색 빛이 감도는 따뜻한 먹물색 배경에 살짝 노이즈가 들어간 불투명 유리(glassmorphism) 질감의 카드가 얹혀 있다. 강조 색상도 예전의 파란색/보라색에서 금색, 비취색, 산호색으로 바뀌었고 글자 크기도 약간 커져 구분이 뚜렷해졌다. 공유 기능에는 '.png 이미지로 저장' 버튼이 새로 생겼는데, 인터넷 연결 없이 완전히 오프라인으로 보고서를 이미지로 만들어준다. '프로젝트 이름 숨기기' 스위치를 켜면 이미지에서도 비밀스러운 이름들이 자동으로 가려진다. 또한, 공유 창의 제목도 더 중립적으로 바뀌어 굳이 직장 상사나 동료에게 보낸다고 단정 짓지 않는다. 보이지 않는 곳에서는 시스템의 보안과 자동화 검사(CI)도 최신 버전으로 튼튼하게 보강했다. 일반 종이에 인쇄되던 흑백 명세서가 금박을 입힌 무광 유리 초대장으로 바뀐 것과 같다. 보기에도 아름다울 뿐만 아니라 친구에게 보여주려고 사진을 찍을 때 버튼 한 번이면 사적인 정보에 알아서 모자이크까지 쳐 준다.
