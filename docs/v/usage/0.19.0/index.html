zh-TW

可以徹底隱藏沒在用的 AI 工具，選單也變簡潔了

如果你只用某一套 AI 工具，現在可以讓畫面更乾淨了。「更換面板」選單裡多了一個「隱藏區塊」的功能，讓你可以自由隱藏 Claude Code 或是 Codex。隱藏之後，不只面板上的卡片會消失，連選單列上的百分比也會跟著不見，讓剩下的那套工具直接排在最前面（如果你把兩套都隱藏了，選單列上還是會保留一個小爪子圖示讓你點擊）。而且不用擔心找不到設定按鈕，當第一張卡片被隱藏時，換面板的按鈕會自動搬家到下一張卡片上。同時，設定選單也變短了，多餘的「自動檢查更新」開關被拿掉（系統預設會在背景自動檢查），隱藏開關也都收合在一起。就像一間工具箱，以前就算你不用扳手，它也硬生生佔著最顯眼的位置；現在你可以把不用的工具收進暗格裡，讓最常用的那把螺絲起子直接擺在最順手的地方。

Original changelog
**Hide Claude Code section**: a new "Hide Sections ▸" submenu in the Switch Panel menu lets you hide Claude Code and Codex independently, so Codex-only users can hide the Claude Code card from every panel theme and the Claude Code percentage from the menu bar (Codex then leads the readout). Every panel keeps its "Switch Panel" button reachable — when the Claude Code card is hidden, the button moves to the next visible card. (#35, requested by @ilss0902)
**Hiding a provider now also hides its percentage from the menu bar** (previously "Hide Codex Section" only hid the popover card). With both providers hidden, the paw icon stays in the menu bar as the click target.
**Shorter settings menu**: the "Automatically Check for Updates" row is gone — update checks simply stay on by default (still honored if disabled in `~/.claude/usage-preferences.json`), and the two hide toggles are consolidated into the "Hide Sections ▸" submenu.

en

Completely hide unused AI tools and enjoy a shorter settings menu

If you only use one AI tool, you can now make the interface much cleaner. The "Switch Panel" menu now features a "Hide Sections" submenu, allowing you to independently hide Claude Code or Codex. Once hidden, not only does the tool's card disappear from the popover, but its percentage also vanishes from the menu bar, bumping your remaining tool straight to the front (if you hide both, a little paw icon remains so you can still click it). You also don't have to worry about losing the settings button; if the first card is hidden, the button automatically moves to the next visible card. At the same time, the settings menu has gotten shorter: the redundant "Automatically Check for Updates" toggle is gone (the system handles it silently by default), and the hide toggles are neatly grouped together. Like a toolbox that used to force a wrench into the spotlight even if you never used it — now you can tuck unused tools into a hidden drawer, keeping your favorite screwdriver right where you need it.

ja

使っていない AI ツールを完全に隠せるようになり、メニューもすっきりした

特定の AI ツールしか使わない場合、画面をさらにすっきりさせることができる。「パネルを変更」メニューに「セクションを非表示」という機能が追加され、Claude Code または Codex を自由に隠せるようになった。隠すと、パネル上のカードが消えるだけでなく、メニューバー上のパーセンテージも一緒に消え、残ったツールが先頭に繰り上がる（両方隠した場合は、クリック用に小さな肉球アイコンだけが残る）。設定ボタンを見失う心配もない。最初のカードが隠れた場合、ボタンは自動的に次の見えるカードへ移動する。同時に設定メニューも短くなり、不要な「アップデートを自動確認」スイッチが削除され（システムがデフォルトで裏で確認する）、非表示スイッチも一つにまとめられた。以前はレンチを全く使わなくても一番目立つ場所を陣取っていた工具箱が、今は使わない工具を隠し棚にしまい、一番よく使うドライバーを一番手の届きやすい場所に置けるようになったようなものである。

ko

쓰지 않는 AI 도구를 완전히 숨길 수 있고, 설정 메뉴도 깔끔해졌다.

특정 AI 도구만 쓴다면 이제 화면을 더 깔끔하게 만들 수 있다. '패널 변경' 메뉴에 '섹션 숨기기' 기능이 추가되어 Claude Code나 Codex를 입맛대로 숨길 수 있다. 숨기면 패널의 카드가 사라질 뿐만 아니라 메뉴 바의 백분율도 함께 사라져, 남은 도구가 맨 앞으로 당겨진다(두 도구를 모두 숨겨도 클릭할 수 있도록 작은 발바닥 아이콘은 남겨둔다). 설정 버튼을 못 찾을까 걱정할 필요도 없다. 첫 번째 카드가 숨겨지면 버튼이 알아서 다음 카드 위로 이사 간다. 동시에 설정 메뉴도 짧아져, 쓸데없던 '업데이트 자동 확인' 스위치가 빠지고(시스템이 기본으로 백그라운드에서 알아서 확인한다) 숨김 스위치들도 한데 묶였다. 예전엔 렌치를 아예 안 써도 공구함의 가장 좋은 자리를 떡하니 차지하고 있었지만, 이제는 안 쓰는 공구를 비밀 서랍에 치우고 가장 많이 쓰는 드라이버를 제일 손 뻗기 좋은 곳에 놓아둔 것과 같다.
