zh-TW

鳳凰動畫恢復生機，不再共用高畫質大圖

選單列上的鳳凰精靈動畫不再像定格畫面一樣死板了。在前一個版本中，系統不小心把選單列的 5 張動畫圖格，全部換成了原本只該給「年度總結報表」使用的高畫質靜態圖片，導致動畫看起來幾乎沒在動。現在選單列已經換回原本活潑的動畫圖檔，而年度報表也改去讀取各自獨立的專屬圖片（`wrapped.png`），兩邊不再共用同一個檔案。就像你不小心把身分證上的大頭照，拿去貼在需要連貫動作的翻頁動畫書裡，不管怎麼翻看起來都像沒動；現在動畫書換回了原本手舞足蹈的連續圖，身分證也保有自己專屬的高清大頭照，兩邊互不干擾。

**Phoenix menu-bar animation was nearly static in 0.25.1**: that release accidentally replaced all 5 menu-bar animation frames with near-identical high-res art meant only for the Year Wrapped report card, so the animation barely changed between frames. The menu-bar frames are restored to the original animated set, and the Wrapped report card now reads from its own dedicated `wrapped.png` asset per critter so the two no longer share a file.

en

Phoenix animation returns to life, decoupled from the high-res report asset

The phoenix spirit animation in your menu bar is no longer frozen like a still photograph. In the previous release, the system accidentally replaced all 5 frames of the menu-bar animation with near-identical, high-resolution static art that was only meant for the "Year Wrapped" report card, making the bird look like it was barely moving. The menu-bar animation has now been restored to its original lively sprite set, and the Wrapped report card now reads from its own dedicated image file (`wrapped.png`). It's like mistakenly pasting your rigid passport photo onto every page of a flipbook animation—no matter how fast you flip it, nothing moves; now, the flipbook gets its dancing sequence back, while the passport keeps its dedicated high-res portrait, completely separate from each other.

不死鳥のアニメーションが復活し、高解像度画像との共有を解除

メニューバーの不死鳥の精霊アニメーションが、一時停止したような不自然な動きから元の活発な姿に戻った。一つ前のバージョンでは、メニューバーの5枚のアニメーション用画像が、本来は「年間レポート」でのみ使われるべき高解像度の静止画像にすべて置き換わってしまっていたため、アニメーションがほとんど動いて見えなかった。今回、メニューバーの画像は元の動きのあるセットに復元され、年間レポート側はそれぞれ専用の独立した画像（`wrapped.png`）を読み込むようになり、両者で一つのファイルを共有することはなくなった。パラパラ漫画のすべてのページに証明写真を貼り付けてしまい、どうめくっても動かないように見えていたのが、今はパラパラ漫画には元の踊るような連続画が戻り、証明写真には専用のきれいな顔写真が使われ、両者が干渉しなくなったようなものである。

ko

불사조 애니메이션이 생기를 되찾고, 고화질 이미지와 파일을 따로 쓴다.

메뉴 바의 불사조 정령 애니메이션이 더 이상 정지 화면처럼 뻣뻣하지 않다. 이전 버전에서는 시스템이 실수로 메뉴 바의 애니메이션 프레임 5장을 모두 '연말 결산 보고서'에만 써야 할 고화질 정지 이미지로 바꿔버리는 바람에 새가 거의 움직이지 않는 것처럼 보였다. 이제 메뉴 바는 원래의 생동감 넘치는 애니메이션 이미지로 되돌아왔고, 연말 보고서 역시 서로 파일을 같이 쓰지 않고 각자 전용 이미지(`wrapped.png`)를 따로 불러오게 되었다. 연속으로 움직여야 할 플립북(파라라락 넘겨보는 애니메이션 책)의 모든 장에 딱딱한 여권 사진을 붙여놓아 아무리 빨리 넘겨도 안 움직이던 것을, 플립북은 다시 춤추는 그림으로 돌려놓고 여권은 자기만의 고화질 사진을 갖게 하여 서로 섞이지 않게 한 것과 같다.
