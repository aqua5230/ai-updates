zh-TW

新增午夜水族箱面板，打包版 App 終於能聽懂你的語言設定

現在多了一款「午夜水族箱（Aquarium）」面板可以選了！這是一個深海動畫主題，裡面有 42 顆不斷往上飄移的氣泡、4 隻透著青色光芒上下浮動的水母，還有微微發光的浮游生物，搭配半透明的毛玻璃卡片與掃光特效，非常療癒。此外，之前因為系統設定的衝突，導致你開啟打包版 App 時，系統會直接無視你的電腦語言設定、硬生生塞給你英文介面；現在我們換了另一種打招呼的方式，繁體中文的使用者點開 App 時，終於能立刻看到熟悉的中文介面了。就像是你買了個智慧魚缸，以前它只會對你講英文，現在它終於學會看你家的門牌，開口用你的母語跟你問好了。

Original changelog
**New "Midnight Aquarium" panel**: sixth built-in panel with a deep-sea animation theme — Canvas 2D bubbles rising from the bottom (42 bubbles with random drift), 4 CSS jellyfish (floating up/down with cyan glow), bioluminescent particles in the background. Glass-morphism cards with backdrop-filter blur, progress bars with a sweeping light animation. Adds i18n key `panel_aquarium` (all 5 languages).
**Fix .app language detection**: switched to `NSLocale.preferredLanguages()` instead of `currentLocale().localeIdentifier()` so the bundle language is no longer overridden by `CFBundleDevelopmentRegion = English` — Traditional Chinese users now see the correct UI language when launching the .app.

en

Midnight Aquarium panel added, and the bundled app finally respects your language settings

You can now choose the new "Midnight Aquarium" panel! It's a deep-sea animation theme featuring 42 randomly drifting bubbles rising from the bottom, 4 jellyfish floating up and down with a cyan glow, and bioluminescent particles, all paired with frosted glass cards and sweeping light effects for a truly soothing vibe. Additionally, due to a conflicting system setting, launching the bundled app previously caused the system to completely ignore your computer's language preference and force an English interface on you. We've changed how we ask the system for your preference, so Traditional Chinese users will finally be greeted with a familiar Chinese interface the moment they click the app. It's like buying a smart fish tank that used to only speak English, but has finally learned to read your house's nametag and greet you in your native tongue.

Original changelog

Glass-morphism cards with backdrop-filter blur, progress bars with a sweeping light animation. Adds i18n key `panel_aquarium` (all 5 languages).
**Fix .app language detection**: switched to `NSLocale.preferredLanguages()` instead of `currentLocale().localeIdentifier()` so the bundle language is no longer overridden by `CFBundleDevelopmentRegion = English` — Traditional Chinese users now see the correct UI language when launching the .app.

真夜中の水族館パネルを追加、パッケージ版アプリが言語設定を正しく反映

ko

한밤의 수족관 패널 추가, 패키지 앱이 드디어 언어 설정을 알아들음

이제 '한밤의 수족관(Aquarium)' 패널을 고를 수 있다! 심해 애니메이션을 주제로 한 이 패널에는 위로 끊임없이 떠오르는 42개의 방울, 청록색 빛을 띠며 위아래로 둥둥 떠다니는 4마리의 해파리, 그리고 은은하게 빛나는 플랑크톤이 반투명한 불투명 유리 카드와 빛이 스쳐 지나가는 효과와 어우러져 무척 마음을 편안하게 해준다. 더불어 예전엔 시스템 설정이 충돌하는 바람에 패키지 앱을 켤 때 시스템이 컴퓨터의 언어 설정을 완전히 무시하고 다짜고짜 영어 화면을 들이밀었지만, 이제는 인사하는 방식을 바꿔 번체 중국어 사용자가 앱을 누르면 마침내 익숙한 중국어 화면을 바로 볼 수 있게 되었다. 예전엔 영어로만 떠들던 스마트 어항을 샀는데, 이제 드디어 당신 집의 문패를 읽고 모국어로 인사할 줄 알게 된 것과 같다.
