發布日期：2026-06-10

zh-TW

更多面板的專案清單變得更乾淨

駭客任務、報紙、Windows 95 這三個面板現在看起來更清爽了。因為這些面板本來就會幫每個專案畫一條用量進度條，系統就把原本夾在中間多餘的列分隔線拿掉了；現在只靠那條用量條來區分專案，跟預設面板一樣乾淨俐落（至於本來就沒有用量條的面板，還是會保留分隔線）。就像一本已經有明顯段落標題的書，現在把段落之間多餘的橫線都擦掉，讓版面不再眼花撩亂。

Original changelog
**Cleaner project list on more panels**: removed the redundant row separators on the Matrix, Newspaper, and Windows 95 panels — each already shows a per-project usage bar, so projects are now divided by that bar alone, matching the default panel. (Panels that rely on separators instead of a usage bar are unchanged.)

en

Cleaner project lists on even more panels

The Matrix, Newspaper, and Windows 95 panels now look much cleaner. Since these panels already draw a usage progress bar for each project, the system has completely dropped the redundant row separators squeezed between them; projects are now cleanly divided by that usage bar alone, just like the default panel (panels that naturally lack a usage bar will still keep their separators). It's like taking a book that already has clear paragraph headings and erasing all the unnecessary horizontal lines between them, making the page much easier to read.

Original changelog
**Cleaner project list on more panels**: removed the redundant row separators on the Matrix, Newspaper, and Windows 95 panels — each already shows a per-project usage bar, so projects are now divided by that bar alone, matching the default panel. (Panels that rely on separators instead of a usage bar are unchanged.)

zh-CN

更多面板的项目清单变得更干净

骇客任务、报纸、Windows 95 这三个面板现在看起来更清爽了。因为这些面板本来就会帮每个项目画一条用量进度条，系统就把原本夹在中间多余的列分隔线拿掉了；现在只靠那条用量条来区分项目，跟默认面板一样干净利落（至于本来就没有用量条的面板，还是会保留分隔线）。就像一本已经有明显段落标题的书，现在把段落之间多余的横线都擦掉，让版面不再眼花缭乱。

Original changelog
**Cleaner project list on more panels**: removed the redundant row separators on the Matrix, Newspaper, and Windows 95 panels — each already shows a per-project usage bar, so projects are now divided by that bar alone, matching the default panel. (Panels that rely on separators instead of a usage bar are unchanged.)

ja

**Cleaner project list on more panels**: removed the redundant row separators on the Matrix, Newspaper, and Windows 95 panels — each already shows a per-project usage bar, so projects are now divided by that bar alone, matching the default panel. (Panels that rely on separators instead of a usage bar are unchanged.)

ko

더 많은 패널에서 프로젝트 목록이 깔끔해졌다.

매트릭스, 신문, Windows 95 등 세 가지 패널이 훨씬 시원해졌다. 이 패널들은 원래 프로젝트마다 사용량 진행 막대를 그려주기 때문에, 그 사이에 끼어 있던 쓸데없는 줄 구분선을 모두 없앴다. 이제 진행 막대만으로 프로젝트를 나누어 기본 패널처럼 아주 깔끔해졌다(원래 진행 막대가 없는 패널은 구분선을 그대로 둔다). 이미 문단 제목이 뚜렷한 책에서 문단 사이의 쓸데없는 가로줄을 몽땅 지워 눈을 편안하게 만든 것과 같다.
