Windows 全面支援：免安裝直接跑，還有專屬系統匣

Windows 使用者現在能獲得與 Mac 同等級的完整體驗，不用再經過繁瑣的設定就能直接查看 AI 的使用額度了。這次更新讓 usage 原生地支援 Windows 系統，包含 TUI（文字使用者介面，沒有滑鼠按鈕、純靠鍵盤在黑底白字命令列裡操作的畫面）、Claude Code 的 status-line hook（狀態列掛鉤，一種能把自己掛在命令列提示字元旁邊、隨時吐出剩餘額度的小程式），以及 Codex 歷史紀錄的解析。你可以在系統右下角看到一個動態的常駐圖示，隨時顯示 Claude 額度百分比，滑鼠懸停還能看見 Claude 與 Codex 兩者的視窗摘要。左鍵點擊會透過 Windows 內建的 WebView2（微軟內建用來在一般軟體裡顯示網頁的底層引擎）開啟那 11 款華麗的 HTML 報告面板；右鍵選單則能切換面板、重新整理、設定開機自動啟動、檢查更新以及結束程式。如果你不想自己安裝環境，GitHub Releases 現在直接提供打包好的 usage-windows.zip，解壓縮點開 usage.exe 就能用（需有 Windows 10/11 通常已內建的 Microsoft Edge WebView2 Runtime）。背後為了這件事，開發團隊的 CI（持續整合，每次開發者交程式碼時，在雲端自動幫忙測試並打包成軟體的機器人流水線）改用 PyInstaller（一種能把需要環境設定的 Python 程式碼，連同工具包全部壓成一個單一執行檔的打包軟體）來製作發布檔，並把原本 Mac 專用的依賴套件隔開。同時，狀態列掛鉤在讀寫檔案時的鎖定機制與指令引號寫法，也都針對跨平台做了調整，確保兩邊都不會卡死。這就像是原本只在蘋果生態系發行的限定版精裝日曆，現在不但為 Windows 使用者重新開模做了一份，還直接升級成插電即用的免安裝電子版，並且附贈了一個隨時能看的精巧小時鐘釘在你的桌面上。

Original changelog
**Full Windows support**: `usage` now runs natively on Windows, including the TUI, Claude Code status-line hook, and Codex history parsing.
**Windows system tray UI**: a dynamic tray icon shows the Claude quota percentage; its tooltip summarizes Claude and Codex windows. Left-click opens the same 11 HTML theme panels used on macOS through WebView2, while the right-click menu offers panel switching, refresh, launch at login, update checks, and quit.
**Portable Windows release**: GitHub Releases now include `usage-windows.zip`, containing `usage.exe` for unzip-and-run use. It requires the Microsoft Edge WebView2 Runtime, which is normally already present on Windows 10 and 11.
Windows packaging and CI now use PyInstaller and `windows-latest` for both checks and release artifacts. The optional `windows` extra installs pystray, pywebview, and Pillow; macOS-only PyObjC dependencies are guarded by a Darwin platform marker.
Status-line hook file locking and generated hook-command quoting are platform-aware, so they work on Windows as well as macOS.

Full Windows support: Native portable release with a dedicated system tray

Windows users can now enjoy a fully native, first-class experience for tracking AI quotas without jumping through complicated setup hoops. The usage tool now runs natively on Windows, fully supporting the TUI (text user interface, a screen operated purely by keyboard in the black-and-white command line without mouse buttons), the Claude Code status-line hook (a small widget that hooks itself next to your command prompt to spit out remaining quotas), and Codex history parsing. You'll find a dynamic system tray icon resting in your lower-right taskbar, displaying your Claude quota percentage; hovering over it summarizes your active windows. A left-click uses WebView2 (Microsoft's built-in engine for rendering web pages inside regular software) to launch the same 11 gorgeous HTML theme panels available on Mac, while a right-click gives you a menu to switch panels, refresh, set it to launch at login, check for updates, or quit. For those who hate setting up environments, GitHub Releases now offers a portable usage-windows.zip — just extract and double-click usage.exe. Under the hood, the CI (Continuous Integration, a robot pipeline in the cloud that automatically tests and packages the software whenever developers submit code) now uses PyInstaller (a packaging software that squashes environment-dependent Python code and its toolkits into a single executable) to build the release, smartly walling off Mac-specific dependencies. Additionally, the file-locking mechanics and command-quoting for the status-line hook have been made platform-aware so they run flawlessly on both operating systems. It's like a premium designer calendar that used to be a Mac-exclusive product — it hasn't just been remolded for Windows, it's been delivered as a fully assembled, plug-and-play digital edition that comes with a handy little clock permanently pinned to your desk.

Windows 全面対応：インストール不要で直稼働、専用のシステムトレイも追加

Windows ユーザーも、面倒な環境構築を経ずに AI の利用枠を直接確認できる、Mac と同等の完全な体験を得られるようになった。今回の更新により、usage は Windows 上でネイティブに稼働するようになり、TUI（テキストユーザーインターフェース、マウスボタンがなく黒地に白文字のコマンドラインでキーボードのみを使って操作する画面）、Claude Code の status-line hook（ステータスラインフック、コマンドプロンプトの横にぶら下がり、いつでも残り枠を吐き出す小さなプログラム）、Codex の履歴解析に完全対応した。画面右下のタスクバーには、Claude のクオータ割合を示す動的なシステムトレイアイコンが常駐し、マウスを重ねると Claude と Codex 両方のウィンドウの要約が表示される。左クリックすると WebView2（通常のソフトウェア内でウェブページを表示するために Microsoft が組み込んでいる基盤エンジン）を介して、Mac と同じ 11 種類の華やかな HTML レポートパネルが開き、右クリックメニューからはパネルの切り替え、更新、ログイン時の自動起動、アップデート確認、そしてプログラムの終了ができる。自分で環境構築をしたくない場合、GitHub Releases で usage-windows.zip の提供が始まり、解凍して usage.exe を叩くだけで直ちに使える（ただし、Windows 10/11 に通常組み込み済みの Microsoft Edge WebView2 Runtime が必要である）。この裏側では、開発チームの CI（継続的インテグレーション、開発者がコードを提出するたびにクラウド上で自動テストとソフトウェアの梱包を手伝うロボットのパイプライン）が PyInstaller（環境設定が必要な Python コードと関連ツールキットを、単一の実行ファイルに圧縮するパッケージングソフト）を使って配布ファイルを生成し、Mac 専用の依存関係を適切に切り離している。同時に、ステータスラインフックがファイルを読み書きする際のロック機構やコマンドの引用符の処理もプラットフォームを識別するようになり、両方の OS でフリーズすることなく機能する。以前は Apple エコシステムでしか発行されていなかった限定版の精装カレンダーが、Windows ユーザー向けに型を作り直されただけでなく、コンセントを挿すだけで使えるインストール不要の電子版に進化し、いつでも見える小さな時計のおまけ付きで机の上に固定されたようなものである。

Windows 전면 지원: 무설치 실행에 전용 시스템 트레이까지

Windows 사용자도 이제 복잡한 설정 없이 AI 사용량을 바로 확인할 수 있는 Mac과 똑같은 온전한 경험을 누릴 수 있다. 이번 업데이트로 usage는 Windows에서 네이티브로 돌아가며, TUI(텍스트 사용자 인터페이스, 마우스 버튼 없이 검은 바탕에 흰 글씨의 명령줄에서 키보드만으로 조작하는 화면), Claude Code의 status-line hook(상태 표시줄 훅, 명령 프롬프트 옆에 매달려 언제든 남은 할당량을 뱉어내는 작은 프로그램), Codex 기록 분석을 완벽하게 지원한다. 시스템 오른쪽 아래 구석에는 Claude 할당량 비율을 보여주는 동적인 시스템 트레이 아이콘이 항상 떠 있고, 마우스를 올리면 Claude와 Codex 양쪽의 창 요약 정보가 뜬다. 왼쪽 클릭을 하면 WebView2(일반 소프트웨어 안에서 웹 페이지를 띄우기 위해 Microsoft가 내장한 기본 엔진)를 통해 Mac에서 쓰던 11개의 화려한 HTML 보고서 패널이 열리며, 오른쪽 클릭 메뉴에서는 패널 전환, 새로고침, 로그인 시 자동 시작, 업데이트 확인, 그리고 프로그램 종료를 할 수 있다. 직접 환경을 설치하기 귀찮다면 GitHub Releases에서 제공하는 usage-windows.zip을 받아 압축을 풀고 usage.exe를 누르기만 하면 된다(단, Windows 10/11에 보통 기본 탑재되어 있는 Microsoft Edge WebView2 Runtime이 필요하다). 이를 위해 개발팀의 CI(지속적 통합, 개발자가 코드를 제출할 때마다 클라우드에서 자동으로 테스트와 포장을 도와주는 로봇 파이프라인)는 PyInstaller(환경 설정이 필요한 Python 코드와 도구들을 통째로 묶어 단일 실행 파일로 압축해 주는 패키징 소프트웨어)를 이용해 배포 파일을 만들고, Mac 전용 의존성 패키지는 따로 분리해 두었다. 동시에 상태 표시줄 훅이 파일을 읽고 쓸 때의 잠금 장치와 명령어 따옴표 표기법도 운영체제를 가리도록 수정하여 양쪽 모두에서 먹통이 되지 않게 했다. 예전에는 애플 생태계에서만 한정판으로 나오던 고급 달력을 Windows 사용자용으로 틀을 새로 짜서 만들어 준 것을 넘어, 콘센트만 꽂으면 바로 켜지는 무설치 전자판으로 업그레이드하고 언제든 볼 수 있는 앙증맞은 시계까지 덤으로 얹어 책상 위에 못 박아준 것과 같다.
