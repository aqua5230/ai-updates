zh-TW

世界盃面板更好看，查不到價格的模型不再假裝免費

許多小地方的顯示變精確了。在「世界盃」面板裡，現在哪邊用量高，哪邊場上的球員就會比較多，而且會壓著對方打；原本兩邊的進度條擠在中間填不滿，現在也能正常從中間往兩邊完整填滿。另外，當系統查不到某個 AI 模型的定價時，以前會顯示誤導人的「$0.00」（看起來像免費），現在會誠實地顯示「—」。而且，Codex 的額度列現在會乖乖照著 Session、每週、每月這種真實時間窗來標示，不再把 30 天的額度誤標成 Session 了。最後，Codex 解析速度變快了，冷啟動時不用再把舊紀錄全部重看一遍。就像你查某件商品的價格，以前查不到系統就直接標 0 元讓你白高興一場，現在會老實畫一條線告訴你它不知道。

Original changelog
**World Cup panel: each team's squad size now reflects its usage**: a side using more of its session quota fields more players (5–8) and presses its formation toward midfield, sharing the same session-percent signal that already drives the ball drift.
**Models with no pricing data show "—" instead of a misleading $0.00**: the cost column rendered unknown models as $0.00, which reads as free rather than unknown; unavailable costs now render as "—" in both the HTML report and the CSV export.
**Codex quota rows are labelled by window length**: the free-plan 30-day window no longer shows as "Session" — each row's label now comes from the window length Codex reports (≈ Session / Weekly / Monthly).
**World Cup quota bars now fill their whole half**: the duel bars used auto-margins that bunched both fills against the centre line, leaving the outer ends blank so a 100% side never reached the track edge; each fill is now anchored to the centre and extended outward.
**Faster Codex log parsing**: unchanged session files are no longer re-read on every poll, and a versioned on-disk cache lets a fresh launch reuse prior parse results instead of re-scanning all history cold.

en

The World Cup panel looks better, and models with no price stop pretending to be free

Several visual details have been sharpened up. In the "World Cup" panel, the side burning more usage will now field more players on the pitch and push their formation forward aggressively; also, the two duel bars that used to get stuck near the middle now properly fill all the way to the edges. Separately, when the system can't find pricing data for an AI model, it used to show a misleading "$0.00" (making it look free), but now it honestly displays a "—". Furthermore, Codex quota rows are now accurately labelled according to their true time window (Session, Weekly, Monthly), so a 30-day quota is no longer mistakenly labelled as a Session. Finally, parsing Codex logs is much faster now, so a fresh launch no longer has to sluggishly re-read your entire history. It's like checking the price tag on an item: the scanner used to ring up $0 and give you false hope when it couldn't find the barcode, but now it simply draws a dash to admit it doesn't know.

월드컵 패널이 더 멋져지고, 가격을 알 수 없는 모델이 더 이상 무료인 척하지 않음

사소한 곳의 표시가 훨씬 정확해졌다. '월드컵' 패널에서는 이제 사용량이 높은 쪽의 선수가 경기장에 더 많이 뛰어나오며 상대 쪽으로 진영을 밀어붙인다. 가운데에 몰려 끝까지 차지 않던 양쪽의 진행 막대도 이제 가운데서부터 양 끝으로 정상적으로 꽉 찬다. 또한 시스템이 특정 AI 모델의 가격을 찾지 못할 때, 예전엔 오해하기 딱 좋게 '$0.00'(마치 공짜인 것처럼)이라 띄웠지만 이제는 솔직하게 '—'를 긋는다. 게다가, Codex의 한도 줄은 이제 Session, 매주, 매월 같은 실제 시간 창에 따라 정확하게 표시되며, 더 이상 30일짜리 한도를 Session으로 잘못 띄우지 않는다. 마지막으로 Codex 기록 분석 속도가 빨라져, 처음 켤 때 옛날 기록을 몽땅 처음부터 다시 들여다볼 필요가 없어졌다. 물건값을 찍어 볼 때 예전엔 가격을 모르면 0원이라고 찍혀 사람을 헛기대하게 만들었는데, 이제는 모른다고 솔직하게 줄을 그어 주는 것과 같다.
