zh-TW

多款主題面板變得更好看，最底下的字不再被切斷

現在切換到特殊主題面板時，畫面不再殘缺或看不清了。駭客任務（Matrix）面板的倒數計時跟狀態文字變亮，不再暗到看不見；黑洞（Black Hole）面板的背景吸積盤跟粒子流不會再被毛玻璃糊成一團，現在能清楚看透。對報紙（Newspaper）與 Win95 面板來說，專案一多，最下面一行的專案甚至底部的文字不會再被悄悄裁掉（原本版面高度計算有誤）。同時，報紙面板現在行距更緊湊，並加上更深的琥珀底色、羅盤浮水印、雙線裝飾框等設計，更有泛黃古紙的味道。就像去主題餐廳，原本有些區塊燈光太暗看不見菜單，或者桌子太小讓最後一道菜掉出桌外，現在不但燈光和桌子都調整好了，裝潢也變得更有沉浸感。

Original changelog
**Matrix panel's countdown and footer text was too dim to read**: `--muted` opacity was bumped up and a subtle glow (matching the card titles) added to the reset countdown and footer status lines.
**Black Hole panel's signature animation was barely visible**: the accretion-disk scene was drawn directly behind the cards, and their 16px frosted-glass blur was smearing it into a faint blob; the blur is now much lighter so the disk, event horizon, and particle stream actually show through.
**Newspaper and Win95 panels could clip the last project row**: both panels' outer layout was missing the `flex: none` / `flex: 1` split every other panel uses to absorb variable content height, so with 3 projects showing, the bottom of the list (and sometimes the footer) got silently cut off by `overflow: hidden`. Layout fixed and registered panel heights recalibrated to the real measured minimum for each panel (Win95 800→870, others tightened after also trimming row spacing).
**Newspaper panel's row spacing was looser than every other panel**: track height, line-heights, and card padding are now in line with Classic's proportions, so the panel isn't noticeably taller without needing extra clipping-avoidance padding.
**Newspaper panel now leans into the "aged paper" look**: deeper amber tone, a faint compass-rose watermark, ink-shadowed titles, and a double-line decorative border.

精簡模式也支援 Codex，一鍵讓兩位助理講話變精簡

你想讓 AI 講話簡短一點時，現在一鍵就能同時設定兩套工具了。精簡模式（Terse Mode，讓 AI 回答變短的開關）現在不只管 Claude Code，也涵蓋 Codex（另一套在終端機運作的 AI 寫程式工具）。只要在選單列開一次這個開關，系統會偵測你電腦上有沒有裝 Codex，有的話就會透過 Codex 原生的掛鉤機制（hooks，讓你在程式啟動時自動跑一段自訂指令的設定檔）幫它裝好對應的規則。不用分開找兩顆開關——一顆開關兩邊都生效。而且當你關掉它時，系統只會拿掉自己裝的那條掛鉤設定，絕對不會動到你自己幫 Codex 裝的其他掛鉤。就像你想讓辦公室裡的兩個助理講話都簡潔一點，以前你要分開個別下指令，現在只要廣播一次開關，兩個人就都會照做，而且當你收回命令時，也不會干擾他們原本在忙的其他工作。

Original changelog
**Terse Mode now covers Codex CLI too**: the same menu-bar toggle installs a matching SessionStart hook for Codex when it's detected on the machine, using Codex's native hooks system (`[features] hooks = true` plus a `~/.codex/hooks.json` entry). No separate switch — one toggle, both tools. Turning it off only removes usage's own hook entry; it leaves the `hooks` feature flag and any other hooks you've installed for Codex untouched.

Terse Mode supports Codex too, making both assistants concise with one click

When you want the AI to speak more concisely, you can now set up both tools simultaneously with one click. Terse Mode (the toggle that makes AI responses shorter) now covers not only Claude Code but also Codex (another AI coding tool running in the terminal). By flipping this single toggle in the menu bar, the system detects if you have Codex installed and automatically sets up the matching rule using Codex's native hooks system (a configuration that runs custom commands automatically on startup). You don't need to hunt for separate switches—one toggle works for both tools. Furthermore, when you turn it off, the system only removes its own hook entry, leaving any other hooks you've manually installed for Codex completely untouched. It's like wanting two assistants in your office to speak more concisely: you used to have to instruct them individually, but now you just make one broadcast and they both comply, and when you rescind the order, it doesn't interfere with their other ongoing tasks.

Original changelog
**Terse Mode now covers Codex CLI too**: the same menu-bar toggle installs a matching SessionStart hook for Codex when it's detected on the machine, using Codex's native hooks system (`[features] hooks = true` plus a `~/.codex/hooks.json` entry). No separate switch — one toggle, both tools. Turning it off only removes usage's own hook entry; it leaves the `hooks` feature flag and any other hooks you've installed for Codex untouched.

여러 테마 패널이 더 예뻐지고, 맨 아래 글자가 잘리지 않는다.

이제 특수 테마 패널로 전환할 때 화면이 잘리거나 흐릿하게 보이는 일이 없다. 매트릭스(Matrix) 패널은 카운트다운과 상태 글자가 밝아져 더 이상 어두워서 안 보이는 일이 없다. 블랙홀(Black Hole) 패널은 배경의 강착 원반과 입자 흐름이 반투명 유리에 뭉개지지 않고 선명하게 비쳐 보인다. 신문(Newspaper)과 Win95 패널의 경우, 프로젝트가 많아지면 맨 아랫줄이나 바닥 글자가 몰래 잘려나가던 문제(이전의 레이아웃 높이 계산 오류)가 고쳐졌다. 아울러 신문 패널은 줄 간격이 촘촘해졌고, 더 짙은 호박색 바탕, 나침반 무늬 워터마크, 두 줄 장식 테두리 등이 더해져 누렇게 바랜 옛날 신문의 느낌을 한껏 살렸다. 테마 식당에 갔을 때 예전엔 조명이 너무 어두워 메뉴판이 안 보이거나 식탁이 좁아 마지막 요리가 상 밖으로 떨어질 뻔했다면, 이제는 조명과 식탁이 제대로 맞춰졌을 뿐만 아니라 인테리어까지 훨씬 몰입감 있게 바뀐 것과 같다.

간결 모드가 Codex도 지원, 클릭 한 번으로 두 조수의 말수를 줄인다

AI가 더 짧게 말하길 원할 때, 이제 클릭 한 번으로 두 도구를 동시에 설정할 수 있다. 간결 모드(Terse Mode, AI의 대답을 짧게 만드는 스위치)는 이제 Claude Code뿐만 아니라 Codex(터미널에서 도는 또 다른 AI 코딩 도구)에도 적용된다. 메뉴 바에서 이 스위치를 한 번 켜면, 시스템이 컴퓨터에 Codex가 깔려 있는지 확인하고 Codex에 원래 있는 훅 시스템(hooks, 프로그램이 켜질 때 사용자 지정 명령어를 자동으로 실행하게 하는 설정 파일)을 통해 알맞은 규칙을 알아서 깔아준다. 스위치를 따로 찾을 필요 없이 하나만 켜면 양쪽 다 적용된다. 게다가 스위치를 끌 때도 시스템이 직접 깔아둔 훅 설정만 지울 뿐, 당신이 Codex에 따로 깔아둔 다른 훅은 절대 건드리지 않는다. 사무실에 있는 두 조수에게 말을 더 짧게 하라고 시킬 때, 예전엔 일일이 지시해야 했지만 이제는 방송 한 번으로 둘 다 따르게 만들고, 명령을 거둬들일 때도 그들이 하던 다른 일에 전혀 방해를 주지 않는 것과 같다.
