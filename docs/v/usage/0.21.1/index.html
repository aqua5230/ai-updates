報告裡現在會用白話文告訴你 AI 工具又更新了什麼

HTML 報告裡現在多了一個「AI 工具更新速報」區塊。它會幫你把你在用的 AI 工具（Claude Code、Codex、Antigravity）近期的更新日誌，翻譯成連外行都看得懂的白話文卡片——每張卡片只講一個重點，而且底下會附上官方的英文原文讓你對照。這個速報是從 GitHub 上抓一個小檔案下來的，所以你的 App 就算沒更新，也能看到最新的工具情報，而且它不會亂連其他網路。就像你訂閱了軟體更新的白話文電子報，它會自己定期送進你的報告裡。

**AI Tool Update Digest in the HTML report**: a new section that sums up recent updates to the AI coding tools you use — Claude Code, Codex, and Antigravity — rewritten in plain, layperson-friendly language as one-idea cards, each keeping the official changelog verbatim underneath. The content is fetched from a small JSON file on GitHub, so it stays current without an app update and makes no other network calls.

en

The report now translates AI tool updates into plain English for you

The HTML report now features an "AI Tool Update Digest" section. It takes the recent changelogs of the AI tools you use (Claude Code, Codex, and Antigravity) and translates them into plain-English cards that anyone can understand — each card focuses on one core idea and keeps the official English original verbatim underneath for reference. This digest is fetched from a tiny file on GitHub, meaning it stays up-to-date even if you haven't updated the app itself, and it doesn't make any other network calls. It's like subscribing to a plain-language newsletter for software updates that automatically delivers itself straight into your report.
