zh-TW

蝶類圖鑑面板在沒有專案時不再亂跳

新增的蝶類圖鑑面板現在不管有沒有專案資料，版面都會穩如泰山。以前這個面板的內容是垂直置中的，當你還沒有任何專案資料時，面板上的卡片會直接飄到視窗正中間，等專案一出現又會突然跳回原本的位置；現在它已經改成跟其他面板一樣靠上對齊，多出來的空間會被專案卡片自然填滿。就像原本掛在牆上的畫因為沒釘好而滑到中間，現在把它牢牢釘在頂部，不管底下多掛幾幅畫都不會走位。

Original changelog
**Lepidoptera panel no longer shifts when the project list is empty**: the panel was vertically centered, so with no project data the cards floated to the middle of the popover and jumped when projects appeared. It now top-aligns like the other panels, with the project card absorbing the extra height, so the layout stays stable whether or not projects are listed.

en

The Lepidoptera panel no longer jumps around when the project list is empty

The new Lepidoptera panel will now stay rock solid regardless of whether you have project data. Previously, this panel was vertically centered, meaning if you had no projects listed, the cards would float to the exact middle of the popover and then suddenly jump back when projects appeared; it now top-aligns exactly like the other panels, with the project card naturally absorbing any extra space. Like a framed picture that used to slide to the middle of a wall because it wasn't nailed properly, it's now firmly anchored at the top and won't shift no matter how many other pictures you hang below it.

나비 도감 패널이 프로젝트가 없을 때 더는 제멋대로 튀지 않는다.

새로 추가된 나비 도감 패널은 이제 프로젝트 자료가 있든 없든 화면이 흔들림 없이 튼튼하다. 예전엔 이 패널의 내용이 세로 가운데 정렬이라서 프로젝트 자료가 하나도 없으면 패널 위 카드들이 창 한가운데로 둥둥 떠 올랐다가, 프로젝트가 나타나면 갑자기 원래 자리로 튀어 오르곤 했다. 이제는 다른 패널처럼 위쪽 맞춤으로 바뀌었고, 남는 공간은 프로젝트 카드가 자연스럽게 채워준다. 벽에 걸어둔 액자가 못을 제대로 안 박아 한가운데로 미끄러지던 것을, 이제 꼭대기에 단단히 박아두어 밑에 그림을 더 걸어도 자리가 틀어지지 않게 만든 것과 같다.
