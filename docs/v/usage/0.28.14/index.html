下一個五小時工作階段可自動接棒

你剛用完 Antigravity 的五小時工作階段（session，一段可使用的固定時段），不想守著倒數等下一輪，現在可以在選單手動開啟自動接棒。它會查看最新、非模擬的 agy 額度結果；只有所選模型群組已完全重設且沒有倒數時，才在背景執行一次本機 agy CLI（命令列工具）的指定指令來開啟下一段。這個選項預設關閉，開過一次後五小時內不會重複觸發，也只使用你電腦上的 agy CLI。像接力賽交棒：跑者真的回到起跑線、計時器歸零才交棒，不會提早衝出去；開啟時也會顯示和 Claude 視窗守護功能相同的 Mac 睡眠提醒。

Original changelog
**Antigravity can now auto-start its next 5-hour session.** A new opt-in menu toggle watches fresh, non-mock agy quota results and, when the selected model group's session is fully reset with no countdown running, dispatches one background `agy -p ok --model 'Gemini 3.5 Flash (Low)'`. It is off by default, self-throttles for five hours, uses only the local agy CLI, and shows the same Mac sleep warning as the Claude window keeper when enabled.

en

Your next five-hour session can start itself

Finished an Antigravity five-hour session and don't want to watch the clock for the next one? You can now opt into an automatic handoff from the menu. It watches fresh, real agy quota results and, only when your chosen model group has fully reset with no countdown running, launches one specified command through the local agy CLI in the background. It is off by default, will not fire again for five hours, and uses only the agy CLI on your computer. Think of a relay race: the baton moves only after the runner is back at the starting line and the clock is at zero, never early. Enabling it also shows the same Mac sleep warning as the Claude window keeper.

ko

다음 5시간 세션이 자동으로 이어서 시작된다

Antigravity의 5시간 세션을 다 쓴 뒤 다음 차례를 기다리기 싫다면, 메뉴에서 자동 이어받기를 선택해 켤 수 있다. 최신의 실제 agy 할당량 결과를 살피고, 선택한 모델 그룹의 세션이 완전히 초기화됐으며 카운트다운도 없을 때만 로컬 agy CLI(명령줄에서 쓰는 도구)로 지정된 명령 하나를 백그라운드에서 실행한다. 기본값은 꺼짐이며, 한 번 실행한 뒤에는 5시간 동안 다시 실행하지 않고, 내 컴퓨터의 agy CLI만 사용한다. 달리기 선수가 출발선으로 돌아오고 시계가 0이 된 뒤에만 바통을 넘기는 계주와 같아서 일찍 출발하지 않는다. 켜면 Claude 창 지킴이와 같은 Mac 잠자기 경고도 표시한다.
