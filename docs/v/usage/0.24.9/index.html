zh-TW

解決報表偶爾罷工的錯誤，徹底擺脫壓縮檔讀取問題

以前看分析報表時，有時候會突然跳出 ZipImportError 這種錯誤說讀不到檔案，現在這問題已經被徹底解決了。這是因為系統在跑報表時，會去讀取三個散落在外面的小檔案，而這些檔案被包在 python313.zip 這種壓縮檔裡，跟之前修好的介面閃退問題是同一個原因。現在工程師把這三個檔案都搬進了報表專用的資料夾裡，讓整個報表產出流程都不用再去拆壓縮檔了。就像是一間工廠的組裝線，原本員工要跑去外面的倉庫，手忙腳亂地拆紙箱拿少數幾個零件，偶爾還會拆壞箱子；現在把所有需要的零件通通搬進組裝線旁邊的零件櫃裡，隨手就能拿到，再也不會出錯卡線了。

Original changelog
**Analysis report occasionally failed with `ZipImportError: bad local file header`**: `analyzer/reporter.py` imported three single-file modules (`persona_loader.py`, `subscription.py`, `ai_updates_loader.py`) that lived outside the package directories py2app already unzips, so they still loaded through `python313.zip` — the same failure mode the earlier adapters/analyzer/ui packaging fix was meant to close. Moved all three under `analyzer/` so the entire report pipeline is now free of zipimport.

en

Fixed occasional report failures by moving all modules out of zipped loads

If you occasionally encountered a ZipImportError that crashed your analysis report, this issue is now completely resolved. The problem happened because the reporting system needed three stray files that were still awkwardly trapped inside the python313.zip archive—the exact same root cause behind an earlier UI crash. The engineers have now moved all three of these files safely inside the dedicated report folder, meaning the entire report pipeline no longer needs to unzip anything to run. It's like a factory assembly line where workers previously had to run to an outside warehouse and frantically tear open cardboard boxes just to grab three specific screws, occasionally ripping the box and stalling the line; now, those screws have been moved to the parts bin right next to the conveyor belt, perfectly within reach.

보고서가 가끔 작동을 멈추는 오류를 해결하고, 압축 파일 읽기 문제에서 완전히 벗어났다.

분석 보고서를 볼 때 가끔 파일을 읽을 수 없다며 ZipImportError가 떠서 보고서가 멈추던 문제가 완전히 해결되었다. 시스템이 보고서를 만들 때 바깥에 흩어져 있는 작은 파일 3개를 읽어와야 했는데, 이 파일들이 python313.zip이라는 압축 파일 안에 갇혀 있어서 벌어진 일이다. 예전에 화면이 갑자기 꺼지던 문제와 똑같은 원인이다. 이제 엔지니어가 이 파일 3개를 모두 보고서 전용 폴더 안으로 옮겨, 보고서를 만들 때 더 이상 압축 파일을 풀 필요가 없게 만들었다. 공장의 조립 라인에서 직원이 예전엔 굳이 밖의 창고까지 뛰어가 종이상자를 허둥지둥 뜯어 부품 몇 개를 꺼내다 가끔 상자를 망가뜨려 라인을 멈춰 세웠다면, 이제는 필요한 부품을 조립 라인 바로 옆 서랍에 모두 옮겨 두어 손만 뻗으면 오류 없이 꺼내 쓸 수 있게 된 셈이다.
