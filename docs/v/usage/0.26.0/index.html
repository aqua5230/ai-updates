zh-TW

Antigravity 正式加入監控行列，三強鼎立

你現在可以同時監看 Antigravity（Gemini）的使用量了，它正式成為與 Claude Code、Codex 並列的第三個追蹤工具。不管你切換到哪種主題介面，都會多出一張專屬的 Antigravity 額度卡片，完全融入當前的設計風格；而且它靠著在背景每 15 分鐘跑一次官方指令來抓額度，不耍暗招也不亂戳內部介面。選單列上除了原本的鳳凰和龍，現在還多了一隻專屬的獅子精靈（spirit companion），牠的動畫快慢會誠實反映你燃燒 Antigravity token 的速度。此外，HTML 深度報表也已經把 Antigravity 的紀錄整合進去。就像是你車上的儀表板原本只能看油量和電量，現在多裝了一顆專屬的水箱溫度表，不但風格完美契合，連指針跳動的速度都跟引擎同步，讓你一手掌握整台車的狀況。

Original changelog
**Antigravity (Gemini) support** — usage now watches a third AI tool alongside Claude Code and Codex:
- **Quota card in every panel**: session + weekly limits appear as a third card in the classic panel and all nine themed quota panels, each styled in its host theme's visual language, with the same stale-data badge and hide toggle as the other two tools.
- **Official numbers, no API poking**: quota is read by periodically running the Antigravity CLI's own `/quota` command in the background (15-minute cache) — identical to typing it yourself. No OAuth-token scraping, no internal API calls.
- **Menu bar segment**: the Antigravity mark and session percentage join the menu bar, complete with a **lion spirit companion** whose animation speed follows Antigravity's own token burn rate — the phoenix (Claude) and dragon (Codex) finally have a third packmate.
- **HTML report integration**: Antigravity token usage from local logs feeds the deep report.

卡片隨你排，說明書多國語言化

現在你可以直接按住拖曳，隨心所欲地把 Claude、Codex 或 Antigravity 的額度卡片換位子了。這份排版順序會自動存下來，就算你切換介面主題或重開機都不會跑掉。另外，當 Antigravity 遇到需要確認信任工作區的提示時，額度探測器（probe）也不會再傻傻卡住，過期資料的提示文字也寫得更準確。專案的 README 說明書現在一口氣支援了五種語言（英、繁中、簡中、日、韓），內容也梳理得更乾淨。就像你可以把自己最常用的悠遊卡、信用卡在皮夾裡隨意換格子，換了新皮夾順序依然照舊；而且買這套皮夾附贈的說明書，現在翻譯成了五國語言，任何人都能看懂。

Fuzz harness imports are static so PyInstaller bundles the parser modules (CI fuzzing only; no user-facing impact).

Antigravity joins the lineup, bringing the big three together

No matter which UI theme you use, a dedicated Antigravity quota card now appears, styled perfectly to match its host theme. It safely reads this quota by running the official CLI command in the background every 15 minutes, with no sketchy token scraping or undocumented API poking. Up in the menu bar, the phoenix and dragon are joined by a new lion spirit companion, whose animation speed accurately reflects how fast you're burning through Antigravity tokens. Finally, Antigravity usage is now fully fed into the HTML deep report. It's like your car's dashboard: it used to only show gas and battery, but now it has a perfectly matching water temperature gauge installed; the needle moves in total sync with the engine, giving you the complete picture at a glance.

Antigravity가 감시 목록에 정식으로 합류하여 3강 체제가 완성되었다.

이제 Claude Code, Codex와 나란히 제3의 도구로서 Antigravity(Gemini) 사용량도 함께 지켜볼 수 있게 되었다. 어떤 테마를 쓰든 Antigravity 전용 할당량 카드가 추가되며, 현재 테마 디자인에 완벽하게 녹아든다. 할당량은 백그라운드에서 15분마다 공식 명령어를 실행해 정직하게 가져오며, 토큰을 훔쳐보거나 내부 API를 마구 찌르는 꼼수는 쓰지 않았다. 메뉴 바에는 원래 있던 불사조와 용에 이어 전용 사자 정령(spirit companion)이 추가되었으며, 이 녀석이 움직이는 속도는 당신이 Antigravity 토큰을 태우는 속도를 정직하게 반영한다. 또한 HTML 딥 리포트에도 Antigravity 기록이 통합되었다. 원래 기름과 배터리만 보여주던 자동차 계기판에 디자인이 똑같은 냉각수 온도계가 새로 달렸고, 바늘 움직임까지 엔진과 완벽하게 동기화되어 차의 상태를 한눈에 꿰뚫어 볼 수 있게 된 것과 같다.
