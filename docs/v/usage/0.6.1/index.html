zh-TW

新增駭客任務面板，標題文字不再被裁切

現在多了一款「駭客任務（Matrix）」面板可以選了！它有著黑底、螢光綠字，還有經典的數位雨動畫。裡面的版面跟預設的面板一模一樣，只要點擊「⇄ 更換面板」按鈕就能切換。同時，我們也把面板標題的行高（line-height，文字的上下空間）稍微拉高，這樣中文或日文標題上的筆畫跟發光效果就不會被卡片邊緣無情地切掉了。就像是幫招牌換了個超酷的霓虹燈新造型，而且還把招牌框稍微做大一點，讓所有的字都能完整透出光芒。

Original changelog
**Matrix panel**: a second built-in panel — black background, neon green type, falling digital rain. Card layout, progress bars, project ranking, and footer all match the Classic panel; only the palette and background differ. Toggle via the `⇄ Switch panel` button in the popover.
README now shows Matrix panel screenshots (Traditional Chinese / English) side-by-side with Classic.
Matrix panel title `line-height: 1` clipped CJK ascenders and the `text-shadow` glow (e.g. `專案用量`, `プロジェクト使用量`) at the card edge; bumped to `1.25` so titles render fully in all five languages and stay vertically aligned with the 30x30 icon.

en

New Matrix panel added, and title text is no longer clipped

You can now choose the new "Matrix" panel! It features a black background, neon green type, and the classic falling digital rain animation. The layout inside is exactly the same as the default panel, and you can switch to it simply by clicking the "⇄ Switch panel" button. At the same time, we slightly increased the panel title's line-height (the vertical space for text) so that the strokes and glow effects on Chinese or Japanese titles are no longer mercilessly clipped at the card's edge. It's like giving a storefront sign a cool new neon look, while also making the frame a bit larger so all the glowing letters fit perfectly.

Original changelog
