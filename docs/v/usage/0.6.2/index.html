zh-TW

駭客任務面板的專案用量圖示回來了

之前在駭客任務（Matrix）面板裡，「專案用量」卡片上的資料夾圖示不見了。這是因為程式碼裡不小心寫了一段無限迴圈的顏色設定（CSS 變數自己參考自己），導致系統不知所措，直接把圖示變成透明的。我們現在把那段多餘的設定移除了，資料夾圖示就正常顯示了。就像是油漆工不小心把顏料罐的標籤貼成了「這罐顏料的顏色」，結果不知道該塗什麼顏色就乾脆不塗，現在把亂標的標籤撕掉，終於塗上正確的顏色了。

Original changelog
**Matrix panel "Project Usage" folder icon missing**: each card carried an inline `style="--accent: var(--accent)"` — a self-referential cyclic CSS variable. Per the CSS spec, cyclic var() resolves to invalid-at-computed-value-time and unsets the property, so the inline SVG's `stroke="var(--accent)"` had no color and rendered transparent. Claude / Codex cards use `<img>` so they were unaffected, but the projects card's inline SVG folder icon disappeared. `--accent` is already defined on `:root` and inherits to all descendants, so the per-card overrides were meaningless — removing them restores the icon.

en

The project usage icon is back on the Matrix panel

Previously, on the Matrix panel, the folder icon on the "Project Usage" card disappeared. This happened because a cyclic color setting (a CSS variable referencing itself) was accidentally written into the code, causing the system to throw its hands up and render the icon completely transparent. We have now removed that redundant setting, and the folder icon displays normally again. It's like a painter accidentally labeling a paint can "the color of this paint" and ending up painting nothing out of confusion; now that the bad label is peeled off, the correct color is finally applied.

マトリックスパネルのプロジェクト使用量アイコンが復活

以前、マトリックス（Matrix）パネルにおいて、「プロジェクト使用量」カードのフォルダアイコンが消えていた。これは、コード内に無限ループとなる色の設定（CSS変数が自分自身を参照するもの）が誤って記述されており、システムが処理できずにアイコンを透明にしてしまったのが原因である。その余計な設定を削除したため、フォルダアイコンは正常に表示されるようになった。ペンキ職人が塗料の缶に誤って「この塗料の色」というラベルを貼ってしまい、何色を塗ればいいか分からず全く塗らなかったのが、でたらめなラベルを剥がしたことでようやく正しい色が塗られたようなものである。

ko

매트릭스 패널의 프로젝트 사용량 아이콘이 돌아옴

예전엔 매트릭스(Matrix) 패널에서 '프로젝트 사용량' 카드의 폴더 아이콘이 보이지 않았다. 코드 안에 무한 반복되는 색상 설정(CSS 변수가 자기 자신을 참조하는 현상)을 실수로 적어 놔서, 시스템이 어찌할 바를 모르고 아이콘을 아예 투명하게 만들어버렸기 때문이다. 이제 그 쓸데없는 설정을 지웠으니 폴더 아이콘이 멀쩡하게 잘 보인다. 페인트공이 실수로 페인트 통에 '이 페인트의 색깔'이라는 이름표를 붙이는 바람에 무슨 색을 칠해야 할지 몰라 아예 안 칠해버렸던 것을, 이제 엉터리 이름표를 떼어내어 제대로 색을 칠하게 된 것과 같다.
