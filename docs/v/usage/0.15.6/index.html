換上全新賽博龐克貓圖示

你安裝或更新 App 後，會發現原本暫用的青色貓掌圖示，已經換成了正式的賽博龐克風格貓咪。現在不管是在 Dock（桌面底部的捷徑列）、Finder 還是頂部選單列，都會顯示這個新圖示。就像店面掛上了正式的招牌，不再用臨時的手寫帆布。

Original changelog
**New cyberpunk-cat app icon**: replaces the teal-paw placeholder with the real usage icon (a cyberpunk-style cat). It ships with the `.app` starting this release, so the new icon shows in the Dock / Finder / menu bar after install.

說明文件更好讀，安裝指令免捲動直接抄

新朋友在看專案首頁的說明文件（README）時會輕鬆很多。現在一打開頁面，最頂端就有一塊「快速上手」區，不用往下捲就能直接複製那行 Homebrew 安裝指令；同時頁面最底下也補上了專案的星星數成長圖表。就像買電器附的說明書，第一頁直接印著「按這裡開機」，不用翻到第三頁才開始用。

Original changelog
**README onboarding improvements**: (1) a top-level Quick Start that lifts the one-line Homebrew install up to where you can copy-paste it without scrolling; (2) a Star History chart at the bottom.

en

Brand new cyberpunk cat app icon

When you install or update the app, you'll notice the temporary teal paw icon has been replaced with the official cyberpunk-style cat. This new icon now shows up everywhere—in your Dock (the shortcut bar at the bottom of your desktop), Finder, and the top menu bar. Like a storefront finally putting up its permanent neon sign instead of a temporary handwritten banner.

Original changelog
**New cyberpunk-cat app icon**: replaces the teal-paw placeholder with the real usage icon (a cyberpunk-style cat). It ships with the `.app` starting this release, so the new icon shows in the Dock / Finder / menu bar after install.

README is easier to read, with install commands ready to copy without scrolling

New users will have a much easier time navigating the project's README file. We've added a "Quick Start" section right at the top, so you can copy the one-line Homebrew installation command without having to scroll down; we also added a Star History chart at the very bottom. It's like an appliance manual that prints "press here to turn on" right on the cover, saving you from flipping to page three just to get started.

Original changelog
**README onboarding improvements**: (1) a top-level Quick Start that lifts the one-line Homebrew install up to where you can copy-paste it without scrolling; (2) a Star History chart at the bottom.

説明書が読みやすくなり、インストール用コマンドもスクロールなしでコピペ可能に

プロジェクトのトップページにある説明書（README）が、新規ユーザーにとってずっと読みやすくなった。ページを開いてすぐの一番上に「クイックスタート」欄を設けたため、スクロールせずとも1行の Homebrew インストールコマンドをコピーできる。また、ページの一番下にはプロジェクトのスター獲得数の推移グラフも追加した。家電の取扱説明書で、3ページ目までめくらなくても表紙に「ここを押して電源オン」と書いてあるようなものである。

설명서가 읽기 편해지고, 설치 명령어는 스크롤 없이 바로 복사할 수 있다

프로젝트 첫 화면의 설명서(README)를 보는 새 사용자들의 부담이 훨씬 줄었다. 이제 페이지를 열자마자 맨 위에 '빠른 시작' 구역이 있어, 아래로 스크롤할 필요 없이 한 줄짜리 Homebrew 설치 명령어를 바로 복사할 수 있다. 페이지 맨 아래에는 프로젝트의 별표(Star) 증가 추이 그래프도 추가했다. 새 가전제품을 샀을 때 3페이지까지 넘길 필요 없이 표지에 바로 '여기를 눌러 켜세요'라고 적혀 있는 설명서와 같다.
