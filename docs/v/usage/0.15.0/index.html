zh-TW

額度快用完時，系統會跳通知提醒你

現在你可以放手工作，不用一直盯著用量看了。只要在選單裡開啟這項功能，當你的 Claude 或 Codex 額度快見底、完全用光、或是重置恢復時，螢幕右上角都會跳出 macOS 系統通知來提醒你。它很克制，每個階段只會提醒一次，不會一直煩你。就像開車時不用死盯著油表，油快沒了儀表板自然會亮燈警告你。

Original changelog
**Quota usage notifications (opt-in, off by default)**: fires a macOS system notification when usage approaches a threshold, runs out, or recovers ("Almost out / Quota is empty / Quota is back"). Covers both session and weekly quotas for Claude Code and Codex; each threshold alerts once and re-arms after the quota resets. Controlled by one menu toggle; notification text is localized across all five languages in `i18n.json`. Triggered from the existing on-disk usage snapshot — **no network, no API calls**. The packaged `.app` now bundles the UserNotifications framework so alerts are delivered.

一眼看出現在燒得比平常快還是省

燃燒警告文字後面，現在多了一個「配速指示器」。這就像開車時油表原本好幾天才會掉一格，現在換成了數位儀表板，每踩一次油門都會告訴你這一秒燒了多少油。這樣你馬上就能判斷，這個飆高的數字只是短暫的波動，還是你平常真的就用這麼兇。

Original changelog
**Pace indicator**: the burn-rate warning line ("at current pace, empty in X") now appends whether you're running some multiple faster than your personal average, or under it — so you can tell at a glance if you're burning hotter than usual.

畫面不再被無意義的額度查詢訊息洗版

在使用 Codex 時，你的視窗不會再莫名其妙被一堆重複的系統訊息塞滿了。以前在某些情況下，Codex 會像學人精一樣，把你剛剛查額度的動作原封不動拋回來，導致舊版程式誤以為那是新訊息，把它們全部印在畫面上洗版；現在系統懂得辨識並過濾掉這些無意義的回音。就像你對著山谷喊話，現在只會聽到別人真正的回答，不會再被自己的回音吵得無法對話。

Original changelog
**Ignore echoed Codex quota queries (#23)**: in some cases Codex echoes a prior quota query verbatim; older versions treated these echoes as new messages and let them flood the window. They're now detected and skipped.

en

Desktop notifications alert you when your quota is running low

You can now focus entirely on your work without constantly checking your usage. If you enable this in the menu, a macOS system notification will quietly pop up in the corner of your screen when your Claude or Codex quota is running low, completely empty, or freshly restored. It's respectful of your attention, only nudging you once per threshold so it never spams you. Like driving a car—you don't need to stare at the gas gauge, because a light will pop on automatically when the tank is getting empty.

Original changelog
**Quota usage notifications (opt-in, off by default)**: fires a macOS system notification when usage approaches a threshold, runs out, or recovers ("Almost out / Quota is empty / Quota is back"). Covers both session and weekly quotas for Claude Code and Codex; each threshold alerts once and re-arms after the quota resets. Controlled by one menu toggle; notification text is localized across all five languages in `i18n.json`. Triggered from the existing on-disk usage snapshot — **no network, no API calls**. The packaged `.app` now bundles the UserNotifications framework so alerts are delivered.

See at a glance if you're burning through tokens faster or slower than usual

Right after the burn-rate warning, you'll now see a new "pace indicator". It's like upgrading your car's fuel gauge from a needle that drops once every few days to a digital dash that updates your instant fuel economy every time you tap the pedal. With this, you'll know right away whether the scary estimate is just a temporary spike, or if that's actually your normal pace.

Original changelog
**Pace indicator**: the burn-rate warning line ("at current pace, empty in X") now appends whether you're running some multiple faster than your personal average, or under it — so you can tell at a glance if you're burning hotter than usual.

The screen is no longer flooded with meaningless repeated quota queries

When using Codex, your window will no longer inexplicably fill up with a wall of repeated system messages. Previously, under certain conditions, Codex would echo your underlying quota queries right back at you, which the older version of the app mistook for new messages and dumped onto your screen; the system is now smart enough to recognize and filter out these meaningless echoes. It's like shouting into a canyon—you'll now only hear genuine replies from others, instead of being deafened by your own echoes bouncing back.

Original changelog
**Ignore echoed Codex quota queries (#23)**: in some cases Codex echoes a prior quota query verbatim; older versions treated these echoes as new messages and let them flood the window. They're now detected and skipped.

消費ペースの警告文の後に、新たに「ペースインジケーター」が追加された。これは車の燃料計が数日に1回目盛りが減るものから、アクセルを踏むたびにその瞬間の燃費を表示するデジタルメーターに変わったようなものである。このおかげで、警告された数字がただの一時的な変動ではね上がっただけなのか、それとも本当に普段からそれほど激しく使っているのかがすぐに判断できる。

意味のない利用枠確認メッセージによる画面の埋め尽くしを解消

Codex を使っている時、画面が不自然に同じシステムメッセージで埋め尽くされることはなくなった。以前は特定の状況下で、Codex が利用枠を確認する裏側の通信をオウム返ししてくることがあり、旧バージョンのアプリはそれを新着メッセージと勘違いして画面に垂れ流していたが、今はこうした無意味なエコーを認識して弾くようになった。谷に向かって大声を出した際、自分のやまびこに邪魔されず、相手の本当の返事だけが聞こえるようになったようなものである。
