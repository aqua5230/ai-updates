zh-TW

額度快用完時，會出現紅字警告

當系統預估你照目前狂用的速度，會在額度重置前先把配額燒光時，原本顯示「重置 X 分鐘」的地方會自動換成一句紅色的警告：「⚠ 剩 X 分用完（重置還要 Y 分）」。當你沒在大量消耗時，面板長得跟平常一模一樣，完全不打擾你。這項警告涵蓋了 Claude Code 與 Codex 的 4 種額度，而且在不同的面板主題上，還會貼心搭配不同風格的紅色。就像是你開車時，平常油表只會安靜指著剩餘油量，但當你猛踩油門、油箱快見底又還沒開到下一個加油站時，它就會立刻亮起紅燈警告你。

Original changelog
**Burn-rate warning**: when usage projects you'll exhaust a quota before the window resets at your current pace, the normal "Resets in X" line is replaced by a red warning: "⚠ Empty in X (resets in Y)". When you're not burning hot, the panel looks exactly the same as before — no extra noise. Covers Claude Code Session / Weekly and Codex Session / Weekly (all 4 quotas), with theme-matched reds on Classic / Matrix / Newspaper / Win95. Internally it samples percent on a 15-minute rolling buffer and projects from the last-10-minute slope; samples are cleared on quota reset to avoid false alarms.

en

Red warning appears when quotas are burning out too fast

When the system estimates that, at your current burning pace, you'll exhaust your quota before the window resets, the normal "Resets in X" text is automatically replaced by a red warning: "⚠ Empty in X (resets in Y)". When you aren't consuming heavily, the panel looks exactly the same as always, causing zero distraction. This warning covers all 4 quotas across Claude Code and Codex, and even sports theme-matched shades of red depending on your chosen panel. It's like driving a car: normally the fuel gauge quietly points to the remaining gas, but when you floor the pedal and are about to run dry before reaching the next gas station, a red light immediately flashes to warn you.

ja

クオータが枯渇しそうな時に、赤字の警告が出るようになった

現在の激しい消費ペースのままではリセット前にクオータ（配額）を使い切ってしまうとシステムが予測した際、本来「リセットまでX分」と表示される部分が自動的に赤い警告文「⚠ 残りX分で枯渇（リセットまでY分）」に切り替わる。大量に消費していない時はパネルは普段と全く同じ見た目で、邪魔になることはない。この警告は Claude Code と Codex の4種類のクオータすべてを網羅しており、異なるパネルのテーマに合わせてそれぞれ異なるスタイルの赤色が用意されている。車を運転している時、普段は燃料計が静かに残量を示しているだけだが、アクセルをベタ踏みして次のガソリンスタンドに着く前にタンクが空になりそうな時には、即座に赤いランプが点灯して警告してくるようなものである。

ko

할당량을 다 써갈 때 빨간색 경고가 나타남

지금처럼 미친 듯이 쓰는 속도라면 할당량이 초기화되기 전에 다 써버릴 것이라고 시스템이 예상하면, 원래 '초기화 X분'이라고 적히던 자리가 빨간색 경고 문구인 '⚠ X분 뒤 다 씀(초기화까지 Y분)'으로 알아서 바뀐다. 평소처럼 얌전히 쓸 때는 패널이 원래 모습 그대로여서 전혀 거슬리지 않는다. 이 경고는 Claude Code와 Codex의 4가지 할당량에 모두 적용되며, 여러 패널 테마에 맞춰 각각 다른 느낌의 빨간색으로 세심하게 칠해진다. 차를 몰 때 평소엔 연료 게이지가 조용히 남은 기름양만 가리키지만, 액셀을 꽉 밟아 다음 주유소에 닿기도 전에 기름통이 바닥나려 하면 즉시 빨간불을 켜서 경고해 주는 것과 같다.
