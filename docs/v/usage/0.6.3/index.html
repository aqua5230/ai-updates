zh-TW

新增視窗 95 面板，而且視窗大小會為面板量身打造

現在多了一款「視窗 95（Windows 95）」面板可以選了！它完美重現了經典的作業系統桌面，有著藍綠色桌布、漸層標題列、灰色 3D 視窗和微凸的塑膠按鈕。同時，系統現在學會了依據每個面板的內容多寡，自動調整彈出視窗（popover）的大小。像視窗 95 面板內容比較精簡，視窗就會自動縮短一些。就像你買衣服，以前不管高矮胖瘦都只能穿均碼，現在每件衣服都會自動幫你修改到最合身的長度。

Original changelog
**Windows 95 panel**: a third built-in panel recreating the classic Windows 95 desktop — teal wallpaper, navy gradient title bars, grey 3D outset windows, chunked segmented progress bars, raised plastic buttons, Tahoma type.
**Per-panel window size**: `HTMLPanel` gains `width` / `height` parameters so each panel can use a popover size that fits its content (default stays 364x812). The Windows 95 panel is more compact and uses 364x768.
README panel section updated to show all three panels side-by-side (Traditional Chinese / English).

en

Windows 95 panel added, and window sizes are now tailor-made for each panel

You can now choose the new "Windows 95" panel! It perfectly recreates the classic operating system desktop, featuring a teal wallpaper, a navy gradient title bar, grey 3D windows, and raised plastic buttons. At the same time, the system has learned to automatically adjust the size of the popover based on how much content each panel has. Since the Windows 95 panel is more compact, its window automatically becomes a bit shorter. It's like buying clothes — previously everyone had to wear one-size-fits-all regardless of their height, but now every piece is automatically tailored to your perfect fit.

윈도우 95 패널 추가, 패널에 맞춰 창 크기도 맞춤 조절
