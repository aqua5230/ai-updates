zh-TW

狀態列進度條換上新裝，一眼看懂額度緊不緊

終端機狀態列上的進度條現在變得更醒目了。以前的進度條是用普通的 ANSI 綠、黃、紅，現在換成了對比更強的 256 色青綠、暖橘和暗紅，而且格子符號也換成了更清晰的「■□」（實心／空心方塊）。現在你只要瞄一眼，就能立刻分辨目前的額度用量是處於安全、警告還是危險狀態。這就像是把原本黯淡的紅綠燈換成了高亮度的 LED 號誌，大老遠就能看清要不要踩煞車。另外，我們也順便把說明文件裡的繁體中文面板截圖換成了包含近期介面改動的最新版本。

Original changelog
**Statusline progress bar visual refresh**: progress bar characters switched from `█░` to `■□` (filled / hollow squares), and the color palette moved from standard ANSI green/yellow/red (32/33/31) to 256-color teal/orange/dark-red (42/214/160) for stronger contrast around the 50% threshold — safe / warning / danger states are now distinguishable at a glance. Changes confined to `usage_statusline.py` (`progress_bar()` and `color_by_pct()`); HTML reports and the TUI progress bars are unaffected.
**Traditional Chinese default panel screenshot refreshed**: `docs/繁體中文面板.png` updated to reflect the latest UI (new "Report / Terminal" toggle, per-project cost display, footer attribution).

en

Statusline progress bar gets a visual refresh for at-a-glance readability

The progress bar on the terminal statusline is now much easier to read. We've switched the characters to clearer solid and hollow squares ("■□"), and moved from standard ANSI colors to a high-contrast 256-color palette (teal, orange, and dark red). Now, you can instantly tell if your usage is in a safe, warning, or danger state with just a quick glance. It's like upgrading a dim traffic light to ultra-bright LEDs—you know exactly when to hit the brakes from a mile away. We've also updated the Traditional Chinese panel screenshot in the documentation to reflect the latest UI changes.

ステータスラインのプログレスバーを刷新し、利用枠の状況が一目でわかるように

ko

상태 표시줄 진행률 막대 새 단장, 한눈에 파악하는 남은 한도

터미널 상태 표시줄에 있는 진행률 막대가 훨씬 눈에 잘 띄게 변했다. 예전에는 평범한 ANSI 기본 색상인 초록, 노랑, 빨강을 썼지만, 이제는 대비가 훨씬 강한 256색 청록, 주황, 진홍색으로 바꾸고 칸 기호도 더 선명한 '■□'(채워진 사각형/빈 사각형)으로 교체했다. 이제 곁눈질만으로도 현재 사용량이 안전한지, 경고 수준인지, 위험한 상태인지 바로 알아챌 수 있다. 희미하던 신호등을 눈부신 LED로 바꿔 멀리서도 브레이크를 밟아야 할지 단번에 알 수 있게 된 것과 같다. 또한, 설명서에 있는 번체 중국어 패널의 스크린샷도 최근의 사용자 인터페이스 변경 사항을 반영하여 최신 버전으로 업데이트했다.
