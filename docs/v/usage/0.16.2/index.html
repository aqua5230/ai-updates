zh-TW

用 Homebrew 安裝不再報錯，App 捷徑自動歸位

現在你能順利透過 Homebrew 安裝這套軟體了。以前用 formula（給純文字終端機指令用的套件格式）安裝這個圖形介面 App 時，會因為壓縮檔解開後路徑重疊跳出 Errno::ENOENT 錯誤而徹底失敗；現在改用 cask（專給 macOS 圖形介面軟體用的套件格式）發佈，不只徹底解決了這個安裝卡死的問題，還會自動把 App 本體放進你的「應用程式」資料夾裡。如果你以前裝過，只要先跑 brew uninstall usage 刪除，再跑 brew install --cask aqua5230/usage/usage 就能重新裝好。就像你原本買家具遇到說明書寫錯導致螺絲鎖死，現在廠商直接派專車把組裝好的沙發送到你家客廳。

Original changelog
**Homebrew now ships as a cask**: usage is a GUI app, so it's now distributed via Homebrew's cask format — it drops `usage.app` straight into your Applications folder and skips the formula relocation/re-signing pass, which also fully fixes the earlier `usage.app/usage.app` doubled-path `Errno::ENOENT` install failure. Install with `brew install --cask aqua5230/usage/usage`; if you previously installed via the formula, run `brew uninstall usage` first, then reinstall. (Thanks @anatolii-maslennikov-improvado for reporting #34)

預設面板文字更銳利，畫面也更乾淨

看著預設的用量面板時，你的眼睛會舒服很多。系統現在改用更銳利的字體平滑技術與標準字體粗細，並把原本多餘的列分隔線拿掉，還修復了專案代幣（token，AI 算錢的字數單位）數字頂部被切掉一小塊的畫面瑕疵。此外，專案名次變成醒目的實心數字徽章（第一名還會用綠色標示），你目前選中的分頁也會亮起清晰的綠色。就像原本有些模糊的舊電視，現在不僅換成了 4K 高畫質面板，還把畫面上多餘的裝飾線條都擦得一乾二淨。

Original changelog
**Sharper, cleaner default panel**: the default menu-bar panel now renders text with crisper font smoothing and standard font weights, shows project rankings as filled number badges (top project highlighted in green), brightens the active tab, drops the redundant row separators, and fixes the slightly clipped top edge on the project token counts.

en

Homebrew installs no longer fail, and the app automatically drops into Applications

You can now smoothly install this software using Homebrew. Previously, installing this graphical App using a formula (a package format meant for text-only terminal commands) would completely fail with a frustrating Errno::ENOENT error due to overlapping folder paths after extraction; now, it is distributed as a cask (a package format dedicated to macOS graphical software), which not only completely fixes this installation freeze, but also automatically drops the App itself straight into your Applications folder. If you've installed it before, just run brew uninstall usage to remove it, and then run brew install --cask aqua5230/usage/usage to reinstall it properly. Like buying furniture with a misprinted manual that causes stripped screws, now the manufacturer just sends a truck to drop a fully assembled sofa right into your living room.

Install with `brew install --cask aqua5230/usage/usage`; if you previously installed via the formula, run `brew uninstall usage` first, then reinstall. Default panel gets sharper text and a cleaner look

Looking at the default usage panel is now much easier on your eyes. The system now uses crisper font smoothing technology and standard font weights, completely drops the redundant row separators, and fixes a visual glitch where the top edge of the project token (the word-count unit AI uses for billing) numbers was slightly clipped. Furthermore, project rankings have become eye-catching filled number badges (with the top project highlighted in green), and your currently active tab lights up in a clear, bright green. It's like replacing a slightly blurry old TV with a crisp 4K display, and wiping away all the unnecessary decorative lines on the screen.

Homebrew 설치 시 더는 오류가 나지 않으며, 앱 바로 가기가 자동으로 제자리를 찾는다.

이제 Homebrew를 통해 이 소프트웨어를 순조롭게 설치할 수 있다. 예전에는 formula(텍스트 기반 터미널 명령어용 패키지 형식)로 이 그래픽 앱을 설치할 때 압축을 풀고 나면 폴더 경로가 겹쳐서 짜증 나는 Errno::ENOENT 오류를 뿜으며 완전히 실패하곤 했다. 이제는 cask(macOS 그래픽 소프트웨어 전용 패키지 형식)로 배포 방식이 바뀌어, 설치가 먹통이 되던 문제를 완전히 고쳤을 뿐만 아니라 '응용 프로그램' 폴더에 알아서 앱 본체를 넣어 준다. 이전에 설치한 적이 있다면 먼저 brew uninstall usage를 실행해 지우고, 다시 brew install --cask aqua5230/usage/usage를 실행하면 제대로 다시 설치할 수 있다. 예전엔 설명서가 잘못되어 나사가 헛도는 가구를 사야 했다면, 이제는 제조사가 다 조립된 소파를 거실 한가운데 직배송해 주는 것과 같다.
