發布日期：2026-05-27

zh-TW

設定檔搬家不留痕跡，終端機指令不再裝聾作啞

你平常不用管的計費快取（cache，暫存資料的地方）現在搬到了專屬的資料夾裡，而且系統會自己讀舊檔案當備用，你完全不用重設。另外，在終端機打 `usage report` 指令時，如果打錯字，以前系統會假裝沒看到直接跑預設畫面，現在它會明確報錯教你怎麼打；查說明文件（`--help`）的速度也變快了。就像你把家裡的工具箱換了個抽屜放，要用的時候自動拿到手；而平常那個問錯問題就沉默的助理，現在終於會主動說「不好意思，你指的是這個嗎？」了。

Original changelog
**Pricing cache moved under `~/.usage/`**: the LiteLLM pricing cache now lives at `~/.usage/pricing_cache.json` instead of `~/.claude/pricing_cache.json`, following the principle that usage-owned state belongs in its own directory. The legacy path stays as a read-only fallback for seamless migration. Thanks @ericweichun.
**Explicit `usage report --help` and unknown-option handling**: previously the CLI silently ignored unknown report options and `--help` still triggered agent detection. Now `--help` returns the help text immediately and unknown options error out cleanly. Thanks @ericweichun.

en

Configs move homes seamlessly, and terminal commands stop playing deaf

The billing cache (where temporary data is stored) has smoothly moved to its own dedicated folder. The system will still read your old file as a backup, so you don't have to lift a finger to migrate. Also, when you make a typo running the `usage report` command in the terminal, instead of silently ignoring you and running the default report, the system now stops and tells you what went wrong. Looking up the help manual (`--help`) is also noticeably faster. It's like moving your toolbox to a new drawer and having your tools automatically show up in your hand when needed; and that assistant who used to stare blankly when asked the wrong question finally speaks up to say, "Sorry, did you mean this instead?"

ko

설정 파일이 흔적 없이 이사하고, 터미널 명령어가 더는 모른 척하지 않는다.

평소 신경 쓸 일 없는 과금 캐시(cache, 임시로 데이터를 저장하는 곳)가 전용 폴더로 이사했다. 시스템이 알아서 옛날 파일을 예비용으로 읽어 들이므로, 당신은 아무것도 다시 설정할 필요가 없다. 또한 터미널에서 `usage report` 명령어를 칠 때 오타를 내면 예전엔 못 본 척 기본 화면을 띄워버렸지만, 이제는 명확히 오류를 내고 어떻게 치는지 알려준다. 설명서(`--help`)를 부르는 속도도 빨라졌다. 집에 있는 공구함 서랍을 바꿨는데도 쓸 때가 되면 알아서 손에 쥐어지는 것과 같으며, 예전엔 엉뚱한 질문을 하면 입을 꾹 다물던 조수가 이제는 먼저 "죄송한데 이걸 말씀하시는 건가요?"라고 말하게 된 것과 같다.
