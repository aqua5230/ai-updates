配額卡片更聰明，桌面版用戶不再滿頭問號

那些只用桌面版的人，不會再對著沒有數字的 Claude 配額卡片發愣，懷疑自己是不是沒裝好了。配額卡片背後仰賴的是 statusLine hook（一種用來攔截並回報狀態的隱藏小程式），但 Claude Code 只有在你打開互動式終端機時才會叫醒它。以前如果你只用桌面版，這個 hook 永遠不會被觸發，就算你的對話紀錄明明一直在增加，卡片還是只會死板地說「找不到狀態檔」，甚至叫你重新安裝。現在卡片會分辨情況了：如果它發現你明明有裝 hook 也有在聊天，只是缺了狀態檔，它會明確告訴你「需要開一次終端機工作階段來同步」，而不是丟給你根本不適用的安裝教學。就像你裝了智慧電表，但都在用別的插座，電表沒感應到；以前它只會叫你把電表拆掉重裝，現在它會告訴你「去插一下這個專用插座同步數據就好」。

Original changelog
**The Claude quota card now explains why it shows no data for desktop-app/headless-only users**: the quota card depends entirely on the statusLine hook, which Claude Code only invokes when rendering an interactive terminal. Users whose usage happens exclusively through the desktop app or other non-TUI surfaces never trigger it, so the card stayed on a generic "status file not found" message even while the hook was correctly installed and Claude Code transcripts kept growing. The card now distinguishes this case — hook installed, transcripts actively updating, status file still missing — and shows a targeted hint that a terminal session needs to run once to sync, instead of the setup-focused message meant for users who never installed the hook at all.

en

Quota card gets smarter, ending confusion for desktop app users

People who exclusively use the desktop app will no longer stare at a blank Claude quota card wondering if they botched the installation. The quota card relies heavily on the statusLine hook (a hidden mini-program that intercepts and reports status), but Claude Code only wakes it up when you open an interactive terminal. Previously, if you only used the desktop version, this hook was never triggered. Even if your transcripts were actively growing, the card stubbornly displayed a generic "status file not found" message and told you to run the setup again. Now, the card recognizes when the hook is installed and you're actively chatting, but the status file is just missing. Instead of giving you irrelevant installation instructions, it gives you a targeted hint that a terminal session needs to run once to sync. It's like installing a smart water meter but using a different sink; previously the meter would tell you to rip it out and reinstall it, but now it just tells you to "go turn on the main faucet once to sync the data."

ja

クォータカードが賢くなり、デスクトップ版ユーザーの混乱が解消

デスクトップ版しか使っていない人が、数字のない Claude のクォータカード（配額カード）を前に「インストールに失敗したのか」と首を傾げることはなくなる。クォータカードの裏側は statusLine hook（状態を傍受して報告する隠しプログラム）に依存しているが、Claude Code は対話型端末を開いた時にしかこれを呼び出さない。以前はデスクトップ版しか使わない場合、この hook が永遠に起動せず、対話履歴（transcripts）が着実に増えているのに、カードは「ステータスファイルが見つからない」と無機質に告げ、再インストールを促すばかりだった。今はカードが状況を判別するようになり、hook がインストール済みで会話もしているのにステータスファイルだけが欠けていると気づけば、的外れなインストール手順を押し付けるのではなく「同期のために一度端末セッションを開く必要がある」と的確に教えてくれる。スマートメーターを取り付けたのに別のコンセントばかり使っていてメーターが反応しない場合、以前なら「メーターを付け直せ」と指示されたのが、今は「一度メインのコンセントを使ってデータを同期してくれ」と教えてくれるようなものである。

할당량 카드가 똑똑해져 데스크톱 앱 사용자의 혼란을 덜어준다.

데스크톱 앱만 쓰는 사람이 숫자 없는 Claude 할당량 카드를 보며 설치를 잘못했나 어리둥절할 일이 없어졌다. 할당량 카드는 statusLine hook(상태를 가로채 보고하는 숨겨진 미니 프로그램)에 의존하지만, Claude Code는 대화형 터미널을 열 때만 이를 깨운다. 예전엔 데스크톱 버전만 쓰면 이 hook이 영영 켜지지 않아, 대화 기록이 쑥쑥 늘어나고 있는데도 카드는 무뚝뚝하게 '상태 파일을 찾을 수 없음'이라며 재설치를 권하기 일쑤였다. 이제는 카드가 상황을 알아채서, hook도 깔려 있고 대화도 하는 중인데 상태 파일만 없는 걸 확인하면 엉뚱한 설치 안내 대신 '동기화하려면 터미널 세션을 한 번 실행해야 한다'라고 콕 집어 알려준다. 스마트 전력계를 달아 놓고 엉뚱한 콘센트만 써서 전력계가 반응하지 않을 때, 예전엔 전력계를 뜯고 다시 달라고 하던 것이 이제는 '데이터를 동기화하려면 메인 콘센트를 한 번 꽂아 쓰라'고 알려주는 것과 같다.
