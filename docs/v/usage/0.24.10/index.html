zh-TW

更新快訊保留歷史紀錄，錯過的更新隨時能回顧

你現在可以在報表裡回顧以前的更新快訊了。過去，系統每次重新整理時都會把舊的資料覆蓋掉，導致錯過的更新就再也找不回來；現在系統會把每次更新像陣列（array，一連串整齊排列的資料結構）一樣通通存起來，把最新的排在最前面。報表的每張卡片底下也多了一個「查看更新歷史」的摺疊區塊，點開就能翻閱以前的紀錄。同時，系統也抓取了這週最新一期的工具更新內容。就像是車站的公佈欄，以前只要貼了新海報，舊的就會被直接撕掉丟進垃圾桶；現在公佈欄改成了一本活頁簿，新海報依然放在最顯眼的第一頁，但只要你想看，隨時可以往後翻閱過去的每一份公告。

Original changelog
**AI Tool Update Digest now keeps history**: previously each refresh of `ai_updates.json` overwrote the prior period, so older updates were lost. The digest now stores every tool's updates as a versions array (newest first) and the report adds a "View update history" collapsible section beneath each card so older periods stay browsable.
**AI Tool Update Digest content refreshed**: Claude Code 2.1.202, Codex 0.143.0-alpha.38, Antigravity 1.0.16 (covering 2026-07-01~07-08).

en

AI Tool Update Digest now keeps a browsable history of past updates

You can now browse through past AI tool updates right in your report. Previously, every time the digest refreshed, it completely overwrote the old data, meaning missed updates were gone forever. Now, the system stores all historical updates securely as an array (a neatly ordered list data structure), showing the newest ones first. Beneath each tool card in the report, there is now a collapsible "View update history" section you can click to browse older periods. Concurrently, the system has fetched the newest batch of updates for this week. It's like a community bulletin board: instead of tearing down the old posters and tossing them in the trash every time a new one is pinned, the board is now a binder. The newest poster sits proudly on the front page, but you can effortlessly flip back through every announcement ever made.

ja

アップデート情報が履歴を残すようになり、見逃した情報もいつでも確認可能に

レポート内で過去のアップデート情報をさかのぼって見られるようになった。これまで、システムが更新されるたびに古いデータが上書きされ、見逃した情報は二度と戻ってこなかった。今は、システムがすべての更新を配列（array、きれいに並んだデータの構造）のように保存し、最新のものを一番上に並べてくれる。レポート内の各カードの下にも「更新履歴を表示」という折りたたみ式のセクションが追加され、クリックすれば過去の記録を閲覧できる。同時に、今週分の最新のツール更新内容も取得された。駅の掲示板が、以前は新しいポスターを貼るたびに古いものを剥がして捨てていたのが、今はバインダー式のファイルに変わり、最新のポスターが一番目立つ1ページ目にありながら、いつでも過去のすべてのお知らせをめくって読めるようになったようなものである。

업데이트 소식이 지난 기록을 보존하여 놓친 내용도 언제든 찾아볼 수 있다.

이제 보고서에서 예전 업데이트 소식도 다시 찾아볼 수 있다. 전에는 시스템을 새로 고칠 때마다 옛날 자료를 덮어써 버려서, 한 번 놓친 소식은 영영 찾을 길이 없었다. 이제는 시스템이 모든 업데이트를 배열(array, 차곡차곡 줄 세워 둔 데이터 구조)처럼 통째로 저장하고 가장 새로운 내용을 맨 앞에 둔다. 보고서의 각 카드 아래에도 '업데이트 기록 보기'라는 접이식 영역이 생겨서, 누르기만 하면 과거 기록을 넘겨볼 수 있다. 또한, 시스템이 이번 주 최신 도구 업데이트 내용도 가져왔다. 기차역의 게시판이 예전엔 새 포스터를 붙이면 낡은 것은 찢어서 쓰레기통에 버렸지만, 이제는 바인더 수첩으로 바뀌어 가장 눈에 띄는 첫 장에 새 포스터를 걸어두면서도 언제든 뒤로 넘겨 지난 공지사항을 모조리 읽어볼 수 있게 된 것과 같다.
