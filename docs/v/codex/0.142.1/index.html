en

Opt-in Windows system proxy support for smoother authentication

In places with strict network controls like corporate offices, you can now log in without getting blocked. The system added opt-in support for Windows system proxies (intermediary servers that forward your network requests); once enabled, it understands PAC, WPAD (scripts and protocols for automatic proxy configuration), static proxies, and bypass rules set in Windows, keeping you from getting stuck at the authentication screen due to blocked internet access. Like visiting a highly secured building where you used to have to find your own way, now, if you opt in, the building's dedicated guide will automatically lead you through all the checkpoints to your destination.

ja

Windows のシステムプロキシ設定に対応し、社内ネットワークでもスムーズに

会社などネットワーク規制の厳しい場所でも、スムーズにログイン認証できるようになった。Windows のシステムプロキシ（system proxy、ネットワーク要求を中継するサーバー）のサポートが追加され、ユーザーが有効にすれば、Windows に設定された PAC や WPAD（プロキシを自動設定するスクリプトとプロトコル）、静的プロキシ、バイパス規則を読み取り、外部ネットに繋がらずログイン画面で立ち往生することがなくなった。警備の厳しいビルを訪問する際、以前は自分で道を探さねばならなかったのが、今は同意さえすればビル専属のガイドがすべての関所を迂回して目的地へ連れて行ってくれるようなものである。

ko

Windows 시스템 프록시 설정을 지원하여 회사 네트워크에서도 막힘 없이

회사처럼 네트워크 통제가 엄격한 곳에서도 이제 순조롭게 로그인 인증을 할 수 있다. Windows 시스템 프록시(system proxy, 네트워크 요청을 대신 전달해 주는 중간 기착지) 지원이 추가되어, 사용자가 기능을 켜기만 하면 Windows에 설정된 PAC, WPAD(프록시를 자동으로 설정하는 스크립트와 규약), 정적 프록시 및 예외 규칙을 알아듣고, 외부 인터넷에 연결하지 못해 로그인 화면에서 막히는 일이 없어졌다. 경비가 삼엄한 건물에 방문할 때 예전엔 알아서 길을 찾아야 했다면, 이제는 동의만 하면 건물 전속 안내원이 모든 출입 통제 구역을 우회해 목적지까지 데려다주는 것과 같다.
