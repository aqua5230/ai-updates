PAC나 WPAD 같은 복잡한 자동 설정도 거뜬히 통과한다. 마치 해외 직구가 완전 자동화된 것과 같다. 상품 안내 책자가 훨씬 화려해지고 원산지도 또렷하게 적혀 있으며, 회사에서 주문해도 택배가 알아서 회사의 엄격한 수발실과 보안 검색대를 통과해 방화벽 밖에 묶이는 일이 없어진 셈이다.

GPT-5.6 가족의 등장, 그리고 디버깅 및 수동 페어링 새 기술

이제 Amazon Bedrock 사용자는 GPT-5.6 계열(Sol, Terra, Luna)을 선택할 수 있으며, 이들이 전력을 다해 추론하는 '최대(max)' 모드도 완벽히 지원한다. 개발자들에겐 애플리케이션 서버 쪽에 현미경 같은 강력한 도구들이 대거 생겼다. 이제 바탕 환경을 바로 들여다보고, 아래로 뻗어나간 모든 하위 대화를 목록으로 뽑아보며, 심지어 과거 대화의 특정 순간에서 새로운 역사를 '가지치기(fork)'해 테스트해 볼 수도 있다. 또한, 실행 중인 백그라운드 프로그램에서 수동으로 페어링 코드를 뽑아내는 codex remote-control pair 명령어까지 추가되었다. 100% 두뇌를 풀가동할 수 있는 천재 직원들을 무더기로 고용한 것과 같다. 동시에 시간을 되돌리는 회중시계를 얻어서 언제든 회의를 멈추고 남의 노트를 몰래 보거나, 10분 전 발언에서 평행 우주를 뻗어내 결과를 확인해 볼 수 있으며, 리모컨을 수동으로 여는 만능 예비 열쇠까지 얻은 셈이다.

도구 자동 검색 및 밑바탕 패키지 보안 취약점 보완

AI가 MCP(외부 도구 연결 규약)를 부를 때, 이제 기본적으로 키워드로 알맞은 도구를 검색해 쓴다. 게다가 ChatGPT가 대신 관리하는 MCP 서버라면 현재 대화(session) 전용 인증을 쓰라고 명확히 지정할 수 있다. 밑바탕 방어선도 튼튼히 다져서, OpenSSL, Hono, fast-uri, quick-xml 같은 핵심 패키지들을 한꺼번에 업데이트해 최근 발견된 보안 구멍들을 싹 막았다. 또한, 개발자들을 위해 UUID7 코드에 대한 설명서를 친절히 추가하고 원격 연결을 테스트하는 최고의 절차도 정리해 두었다. 당신의 맥가이버칼이 이제 알아서 상황에 맞는 칼날을 뽑아주고, 손잡이에 헐겁던 나사를 조여 사고를 막아주며, 마지막으로 상세한 손질 설명서까지 챙겨주어 언제 쓰든 매끄럽고 안전하게 만들어준 것과 같다.

Windows 터미널 입력 오류 해결, 끊김과 버벅거림을 싹 고친 버그 잡기 대작전

Windows 터미널(ConPTY)을 쓸 때 줄바꿈과 백스페이스 키가 제멋대로 놀던 문제가 드디어 고쳐졌다. 화면에 떴다가 이미 취소된 '보안 경고'가 끈질기게 남아 시스템이 멈춘 것처럼 보이게 만들던 현상도 사라졌다. 원격 서버가 잠깐 끊어졌을 때의 복구 능력도 좋아져서, 미친 듯이 인증 코드 새로 고침을 시도하며 폭주하지 않는다. 또한 시스템이 꺼질 때 미처 다 출력되지 못한 실시간 대화의 꼬리 부분도 이제 허공으로 사라지지 않고 제대로 저장된다. 그뿐만 아니라 시스템이 WebSocket 요청을 비교할 때 중요하지 않은 부가 데이터는 무시할 줄 알게 되어 연결 성공률이 높아졌고, 설치 프로그램도 한 번 받은 배포 정보를 다시 써먹을 만큼 똑똑해져서 툭하면 GitHub의 다운로드 제한에 걸리는 일이 없어졌다. 수리공이 키보드에 낀 백스페이스 키를 고쳐주고, 모니터에 남은 낡은 경고 스티커를 떼어주며, 인터넷이 끊길 때 공유기가 미친 듯이 비프음을 내지 않게 만들고, 컴퓨터가 꺼지기 전 마지막으로 친 문장까지 무사히 저장되게 해준 것과 같다.
