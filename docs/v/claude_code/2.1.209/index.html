zh-TW

背景任務不再擋選單，互動指令恢復正常

現在你在跑代理的背景任務時，又可以順暢使用切換模型等互動選單，不會再被莫名擋下。以前，一個設定得太嚴格的防護機制（guard，用來阻擋非預期操作的保安設定）不小心封殺了 /model 指令與其他對話框，導致你在跑 claude agents（Claude 同時開多個分身代理幫你做事的功能）背景任務（background sessions，在主畫面背後默默執行的工作）時無法叫出這些互動介面。現在這個管太寬的限制已經被撤銷，讓你恢復應有的操作權限。就像辦公室換了新的電子鎖，結果設定太嚴格，連正常上班的員工都被鎖在自己的會議室外面；現在門禁規定撤回原本的標準，你總算能正常推開門辦公了。

en

Interactive menus are no longer blocked in agent background sessions

You can once again use interactive menus, like changing AI models, without them getting stuck while running agent tasks behind the scenes. Previously, an overly strict guard (a protective mechanism meant to prevent unwanted actions) accidentally blocked the /model command and other dialogs from appearing in claude agents (a feature where Claude opens multiple agent instances to work for you) background sessions (tasks running silently in the background). This meant you couldn't interact with these interfaces when you needed to. We've rolled back this overly broad restriction so everything works normally again. It's like an office security door that was set so strictly it locked regular employees out of their own meeting rooms; the rule has been reverted to its standard setting so you can actually push the door open and get to work.

ja

背景タスクでメニューがブロックされなくなり、対話コマンドが正常化

エージェントの背景タスクを実行している最中でも、モデル切り替えなどの対話メニューが再びスムーズに使えるようになった。以前は、厳しすぎる防御メカニズム（guard、予期せぬ操作を防ぐための保安設定）が誤って /model コマンドやその他のダイアログをブロックしてしまい、claude agents（Claude が複数のエージェントを同時に開いて作業を代行する機能）の背景タスク（background sessions、画面の裏側で黙々と動く作業）においてこれらの対話インターフェースを呼び出せなくなっていた。現在はこの広すぎる制限が撤回され、本来の操作権限が回復している。オフィスに新しい電子錠を導入したら、設定が厳しすぎて普通の社員まで会議室から締め出されてしまったが、元の基準に戻したことでようやく普通にドアを開けて仕事ができるようになったようなものである。

ko

백그라운드 작업에서 메뉴가 막히지 않고, 대화형 명령어가 정상화됨

에이전트의 백그라운드 작업을 돌릴 때 모델 변경 같은 대화형 메뉴를 막힘없이 다시 쓸 수 있다. 예전에는 지나치게 엄격한 방어 메커니즘(guard, 원치 않는 조작을 막기 위한 보안 설정)이 실수로 /model 명령어와 기타 대화 상자를 차단하는 바람에, claude agents(Claude가 여러 에이전트를 동시에 열어 작업을 대신 해주는 기능) 백그라운드 작업(background sessions, 화면 뒤에서 조용히 돌아가는 작업) 중에는 이런 대화형 인터페이스를 부를 수 없었다. 이제는 이 지나치게 넓은 제한이 취소되어 원래의 조작 권한을 되찾았다. 사무실에 새 전자 자물쇠를 달았더니 설정이 너무 엄격해 평범한 직원들마저 회의실 밖으로 쫓겨났다가, 이제 출입 규정을 원래대로 되돌려 마침내 정상적으로 문을 열고 일할 수 있게 된 것과 같다.
