zh-TW

內建萬事通導遊，排版與警告訊息更清楚

現在系統裡多了一個叫 antigravity_guide 的內建技能，就像是個隨傳隨到的導遊，當你對 CLI 終端機介面、IDE 或 SDK 有任何疑問時，它可以立刻為你翻找說明書並提供解答。畫面排版也升級了，文件裡的標題和段落間距現在變得更乾淨整齊；系統出錯或給警告時，也會改用專門的「警告訊息」樣式跳出來，不會再跟普通的指令輸出混在一起。如果遇到問題想除錯，/help 選單裡也直接列出了日誌檔（log）的存放位置，不用再自己瞎找了。就像你去博物館，以前只能自己看破舊的解說牌，現在不但拿到了一本排版精美的新導覽手冊，還配了一個專屬解說員，而且緊急廣播也不會再被一般的背景音樂蓋過了。

Original changelog
Added `antigravity_guide` builtin skill to provide instant, in-context reference guides for the Antigravity 2.0, CLI, IDE, and SDK.
Improved markdown rendering by upgrading `glamour` to v2.0.1 for cleaner headings and block padding.
Added alert message type for system errors/warnings, separating them from standard command output.
Added the CLI log file path to the `/help` menu for easy troubleshooting.

瀏覽 Git 歷史更順利，看圖認版本不再卡卡

在終端機裡看程式碼的版本歷史變得更舒服了。以前看 Git 歷史時只有純文字列表，現在它會畫出漂亮的 ASCII 樹狀圖（就是用線條畫出分支合流的圖），讓你一眼看懂版本的來龍去脈。當你捲動歷史紀錄時，旁邊會立刻載入並秀出這個版本改了哪些檔案、改了什麼內容，不用再另外點開。而且系統現在變聰明了，就算你只給它前 6 碼的短版本代號（hash），它也能瞬間認出你指的是哪個長達 64 碼的完整版本。就像看家族族譜，以前只有一堆寫著名字的抽屜，現在不但畫出了清晰的血脈關聯圖，而且你只要拉開抽屜，裡面的人的身世背景就會立刻攤開給你看，就算你只喊他們的小名也能準確找到人。

Original changelog
Improved commit history navigation: scrolling now immediately loads and displays changed files and diffs.
Improved Git integration by enabling ASCII node graphs (`git log --graph`) for visual parity with hg/jj.
Improved commit hash matching to seamlessly resolve short (6-char) to long (64-char) hashes via prefix comparison.

解決權限鬼打牆與 Windows 登入卡關問題

已同意的指令不會再反覆跳權限詢問。以前你明明同意了某個指令權限，但因為系統沒把指令裡的特殊符號（像 $ 或 .）當作普通文字處理，導致它一直認不出你同意過，不停跳出警告鬼打牆；現在這些符號都會被正確辨識了，而且你在設定裡選了「每次都要問」的權限也不會在更新設定檔時被偷偷刪掉。對於 Windows 使用者，現在需要登入時，系統會自動幫你順利叫出瀏覽器畫面，不用再手動複製網址；在執行指令時也不會再遇到因為符號沒處理好導致輸出被默默吞掉的狀況，預設也會改用 PowerShell 來執行了。另外，這次也改善了對 ARM64 設備（像是樹莓派 Raspberry Pi 4B）的相容性。就像去銀行辦事，以前你明明蓋了章，行員卻因為你的印章花紋太複雜而一直叫你重蓋，現在機器終於換新了，不但能精準辨識印章，甚至幫你把本來卡住的號碼牌機也修好了。

Original changelog
Improved compatibility with a broader set of ARM64 devices (e.g. raspberry pi 4b).
Improved authentication to automatically launch browser sign-in via `rundll32`.
Fixed a bug where "ask" permissions were dropped during settings updates, ensuring `settings.json` preservation.
Fixed permission engine matching bugs by escaping regex metacharacters (like `$` or `.`) in saved rules, preventing infinite prompt loops.
Fixed environment flag parsing to prevent ignored disablement flags.
Fixed bash mode argument escaping (preventing swallowed stdout) and defaulted shell resolution to PowerShell.

Built-in guide skill with cleaner markdown and explicit alert messages

There is now a built-in skill called antigravity_guide acting as your personal on-call tour guide; whenever you have questions about the CLI, IDE, or SDK, it can instantly dig through the manuals to provide answers. The layout has also been upgraded: headings and paragraph spacing in documents are much cleaner now. When the system encounters errors or warnings, it now uses a dedicated "alert message" style to pop up, ensuring they no longer blend in with standard command output. If you run into issues and need to troubleshoot, the /help menu now explicitly lists the exact path to your log files so you don't have to hunt them down. Like visiting a museum: you used to have to read tattered old signs by yourself, but now you not only get a beautifully typeset new guidebook, you also get a dedicated human guide, and emergency announcements are no longer drowned out by the background music.

Smoother Git history browsing with visual graphs and smart hash resolution

Browsing code version history in the terminal is now a much more pleasant experience. Previously, viewing Git history just gave you a plain text list; now, it draws beautiful ASCII node graphs (using lines to show branching and merging), letting you understand the flow of versions at a glance. As you scroll through the history, the side panel immediately loads and displays exactly which files were changed and what the diffs are, without requiring extra clicks to open them. Moreover, the system is smarter now: even if you only give it a short 6-character version hash, it instantly resolves it to the full 64-character version you meant. Like looking through a family tree: instead of just staring at a wall of drawers with names, you now get a clear diagram of bloodlines, and simply pulling a drawer instantly reveals that person's entire life story right beside it — and you can accurately find anyone just by calling out their nickname.

Fixes infinite permission loops and Windows login hurdles

Several blood-pressure-raising system issues have been ironed out. Previously, even if you explicitly approved a command's permission, the system would fail to treat special characters (like $ or .) as plain text, constantly failing to recognize your approval and trapping you in an endless loop of permission prompts; these characters are now correctly escaped, and permissions marked "ask" won't secretly get deleted when your settings file updates. For Windows users, when authentication is required, the system now automatically and smoothly pops open your browser, saving you from manually copying URLs. Also, command outputs on Windows will no longer be silently swallowed due to poor argument escaping, and PowerShell is now the default shell. Finally, compatibility with ARM64 devices (like the Raspberry Pi 4B) has been improved. Like dealing with bank paperwork: you used to clearly stamp your seal, but the clerk kept making you re-stamp it because the pattern was "too complex"; now they've upgraded their scanner to recognize it perfectly, and they even fixed the broken queue ticket machine while they were at it.

Git 기록 보기가 훨씬 매끄러워지고, 그림으로 버전을 확인하며 짧은 이름도 잘 알아본다.

권한 확인이 무한히 맴도는 문제와 Windows 로그인 막힘 현상 해결

사람 뒷목 잡게 만들던 시스템 문제 몇 가지를 고쳤다. 예전엔 당신이 어떤 명령어의 권한을 분명히 허락했는데도, 시스템이 명령어 안에 든 특수 기호($나 . 같은 것)를 평범한 글자로 다루지 못한 탓에 허락받은 줄도 모르고 끝없이 경고 창을 띄우며 맴돌았다. 이제 이런 기호들을 제대로 알아채며, 설정에서 '매번 묻기'로 해둔 권한이 설정 파일을 업데이트할 때 몰래 지워지는 일도 없다. Windows 사용자의 경우, 이제 로그인이 필요할 때 시스템이 알아서 브라우저 창을 띄워주므로 인터넷 주소를 일일이 복사할 필요가 없다. 명령어를 칠 때 기호 처리를 잘못해서 출력 내용이 소리 없이 먹혀버리는 문제도 사라졌고, 기본 셸도 PowerShell을 쓰도록 바뀌었다. 또한 ARM64 기기(라즈베리파이 4B 등)와의 호환성도 좋아졌다. 은행에 일을 보러 갔을 때, 예전엔 도장을 제대로 찍었는데도 도장 무늬가 복잡하다며 직원이 계속 다시 찍으라고 우기더니, 이제는 기계가 새 걸로 바뀌어 도장을 찰떡같이 알아보고 내친김에 고장 났던 번호표 기계까지 싹 고쳐놓은 것과 같다.

Original changelog
Improved compatibility with a broader set of ARM64 devices (e.g. raspberry pi 4b).
Improved authentication to automatically launch browser sign-in via `rundll32`.
Fixed a bug where "ask" permissions were dropped during settings updates, ensuring `settings.json` preservation.
Fixed permission engine matching bugs by escaping regex metacharacters (like `$` or `.`) in saved rules, preventing infinite prompt loops.
Fixed environment flag parsing to prevent ignored disablement flags.
Fixed bash mode argument escaping (preventing swallowed stdout) and defaulted shell resolution to PowerShell.
