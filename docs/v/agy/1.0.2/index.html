zh-TW

畫面資訊更貼心，狀態列與說明選單升級

終端機畫面現在更照顧你的隱私與閱讀體驗了。如果你不想讓別人在你背後偷看你的帳號信箱或方案等級，現在可以用 AGY_CLI_HIDE_ACCOUNT_INFO 參數把畫面頂部的這些私人資訊藏起來。在看歷史對話時，你自己打的快捷指令現在前面會多一個大於符號（>），讓你一眼分出哪句是你打的、哪段是 AI 講的。用 /statusline 自訂狀態列時也變得更聰明，現在它聽得懂開關與重設的指令（像 on/off 或 reset），而且不會再因為看說明文件而卡死。打開 /help 選單時，裡面的快捷鍵終於照著按鍵順序排好，也把漏掉的快捷鍵補上了，現在不管是看指令還是看快捷鍵，都可以順暢地整頁捲動。另外，當你用 /diff 想看某個特定的 Git 歷史版本時，它現在終於認得標準的 7 碼短代號了。就像你的辦公桌，以前抽屜亂七八糟、別人還能一眼看到你的薪水單，現在不但加上了密碼鎖，文件也分門別類排好，連標籤上的縮寫也能精準對應。

Original changelog
Added `AGY_CLI_HIDE_ACCOUNT_INFO` environment variable to hide email and plan tier from the header.
Fixed command rendering in message history: prefixed slash commands with a caret (`>`) in response block headers to clearly distinguish user-typed commands from agent outputs.
Fixed statusline subcommand handling and recursive loops: added case-insensitive subcommand parsing (help, delete, reset, enable/on, disable/off) to the /statusline command, providing direct control to toggle or revert custom statuslines and blocking recursive shell hangs during help queries.
Improved `/help` shortcuts tab by sorting shortcuts by keybinding key, adding missing keybindings (like `ctrl+r`, `ctrl+o`, `alt+j`, `ctrl+k`), and generalizing scrolling (PageUp/PageDown/GoToTop/GoToBottom) for both Commands and Shortcuts tabs.
Fixed Git short-hash support in diff selection: updated the commit hash recognition pattern in the  /diff  commit selection tree to match Git's standard 7-character short hashes (and up to 40-character full hashes).

解除 60 秒強迫斷線魔咒，外掛與技能不再迷路

主代理不再被 60 秒互動逾時強制中斷。以前系統有一個預設的「60 秒互動超時」限制，本來是為了管教子代理不要掛網太久，結果不小心連主代理（你正在對話的對象）也被波及，導致它有時講話講到一半就被強制斷線；現在這個緊箍咒只會限制子代理了。在沙盒模式（sandbox）底下抓取網址內容時，系統不會再因為碰到空資料而當機。另外，外掛與技能的路徑也全數導正了：如果你安裝了新的外掛，它現在會乖乖住進共用的設定資料夾（~/.gemini/config/），讓你一裝完就能立刻用；即使在缺乏標準設定資料夾的獨立模式下執行，系統也能順利找到備用的技能檔案，並且不會重複載入。就像修理了一台老是不聽話的機器，以前煮飯超過 60 秒就會被強制斷電，買回來的新外掛也老是被丟進雜物間找不到，現在它不但能煮到熟，新買的配件也能一裝上就立刻發揮作用。

Original changelog
Fixed timeout overrides: restricted the default 60-second interaction timeout specifically to subagents, preventing the main agent from being unconditionally capped.
Fixed a nil-pointer panic in Sandbox Mode: resolved a typed nil interface comparison when fetching URL content.
Fixed fallback skill discovery in Standalone mode: ensures custom/fallback skills are successfully loaded even if the standard configuration directory is missing, and added automatic path deduplication to prevent duplicates.
Fixed plugin installation path mismatch: updated the `plugin` subcommand to install downloaded plugins directly to the shared configuration directory (`~/.gemini/config/`) rather than the private application data folder, making them instantly discoverable.

en

Cleaner UI with privacy options, robust statusline controls, and a better help menu

The terminal interface is now much more mindful of your privacy and reading experience. If you don't want someone looking over your shoulder to see your email address or subscription tier, you can use the AGY_CLI_HIDE_ACCOUNT_INFO environment variable to hide this private info from the top header. When reviewing message history, your typed slash commands are now prefixed with a caret (>), making it instantly obvious which lines you typed and which parts the AI generated. Customizing your status line with /statusline is also smarter now: it understands direct commands to toggle or reset it (like on/off or reset), and no longer hangs the program if you ask for help. When you open the /help menu, the shortcuts are finally sorted alphabetically, missing keys have been added, and you can now smoothly scroll through both the commands and shortcuts tabs using page-up/down keys. Finally, when using /diff to inspect a specific Git version, it now correctly recognizes standard 7-character short commit hashes. Like organizing your office desk: your drawers used to be a mess and anyone could see your pay stub lying around; now everything is locked up, documents are perfectly categorized, and even the shorthand labels are perfectly matched.

Agent timeouts fixed, sandbox panics resolved, and plugins install correctly

We've fixed a few underlying issues that were seriously holding the system back. Previously, there was a default "60-second interaction timeout" limit intended to keep background subagents from hanging indefinitely, but it accidentally affected the main agent (the one you chat with directly), sometimes brutally cutting it off mid-sentence; this strict collar now only applies to subagents. When fetching URL content in sandbox mode, the system will no longer crash if it encounters empty data. Additionally, plugin and skill paths have all been corrected: when you install a new plugin, it is now properly placed in the shared config folder (~/.gemini/config/) so it's instantly usable; and even when running in a standalone mode missing the standard config directory, the system can still successfully locate fallback skill files without loading them twice. Like fixing a constantly misbehaving appliance: it used to forcefully cut the power if it cooked for more than 60 seconds, and newly bought attachments would always get lost in the junk drawer; now, it cooks until it's done, and new attachments work perfectly the moment you plug them in.

터미널 화면이 당신의 사생활과 읽는 경험을 한층 더 배려한다. 다른 사람이 뒤에서 당신의 이메일 계정이나 요금제를 훔쳐보는 게 싫다면, 이제 AGY_CLI_HIDE_ACCOUNT_INFO 변수를 써서 화면 맨 위의 개인 정보를 숨길 수 있다. 과거 대화 기록을 볼 때, 당신이 친 단축 명령어 앞에는 꺾쇠 기호(>)가 붙어 어떤 게 당신이 한 말이고 어떤 게 AI가 한 말인지 한눈에 알 수 있다. /statusline으로 상태 줄을 꾸밀 때도 똑똑해져서, 이제 켜고 끄거나 처음으로 돌리는 명령어(on/off나 reset 등)를 알아들으며 도움말을 찾다 멈춰버리는 일도 없다. /help 메뉴를 열면 드디어 단축키가 키보드 순서대로 가지런히 정리되어 있고 빠졌던 단축키도 채워졌으며, 명령어든 단축키든 휠을 넘기며 편하게 볼 수 있다. 덧붙여 /diff로 특정한 Git 버전을 보고 싶을 때, 이제 시스템이 표준 7자리 짧은 번호(해시)를 제대로 알아본다. 예전엔 서랍이 뒤죽박죽이고 남들이 내 월급명세서까지 빤히 볼 수 있던 사무실 책상에 비유하자면, 이제 자물쇠도 달고 서류도 종류별로 딱딱 나눠 놓았으며 라벨의 줄임말까지 찰떡같이 알아듣는 셈이다.
